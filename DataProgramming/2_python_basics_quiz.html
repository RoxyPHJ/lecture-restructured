Q23객관식⭐⭐⭐[응용]Slide 36–37

❌ X (거짓)

Q25O / X⭐⭐[응용]Slide 30
다음 코드 실행 결과에 대한 문장의 참/거짓을 판별하라.
i = 1 while i < 5: i += 1 if i == 3: break print(i) else: print("done")

"출력은 2 한 줄과 done 한 줄, 총 두 줄이다."
⭕ O (참)
❌ X (거짓)
정답: X (거짓)
실행 추적:
i=1 → i+=1 → i=2, 2≠3 → print(2)
i=2 → i+=1 → i=3, 3==3 → break
else는 while 조건이 자연 소진될 때만 실행. break로 탈출하면 else 블록은 실행되지 않는다.

따라서 출력은 2 한 줄뿐. "done"은 출력되지 않으므로 거짓.
