Introduction — 학습이란 무엇인가?

🌳 나무 예시 (What is a Tree?)

  • 3살 아이에게 사진을 보여주면 나무인지 정확히 답한다
  • 그러나 30살 어른에게 나무의 정의를 물으면 불명확한 답변이 나온다
핵심 통찰: 우리는 나무를 수학적 정의로 배운 것이 아니라, 수많은 나무를 보는 경험(데이터)을 통해 학습했다.

📚 Learning from Data의 3조건

  • 데이터가 존재한다 (We have data)
  • 패턴이 존재한다 (A pattern exists)
  • 수학적으로 정형화하기 어렵다 (Cannot formalize mathematically)
해석적 해(analytic solution)가 없을 때, 데이터로부터 경험적 해(empirical solution)를 구성한다.

🎬 영화 추천 시스템

  • Amazon.com: 매출의 20%가 추천에서 발생
  • Netflix: 10% 개선 = 100만 달러 가치
  • 시청자의 평가 기준은 매우 복잡 → 명시적 모델링 불가
  • 과거 평점 데이터만으로도 충분한 패턴 추출 가능

🎮 인간 수준 게임 플레이

  • Deep Reinforcement Learning (Google / DeepMind)
  • DQN (Deep Q-Network): Atari 게임 수십 종에서 인간 수준 달성
  • AlphaGo: 바둑 세계 챔피언 이세돌 꺾음

🚗 자율주행 차량

  • LIDAR: 360° 3D 지도 생성
  • Stereo Camera: 2D 이미지 + 객체 감지
  • RADAR: 거리 측정
  • GNSS: GPS 위치 산출

📐 수학적 형식화 (Formalization)

  • X = ℝᵈ: 입력 공간 (d차원 유클리드 공간)
  • x ∈ X: 입력 벡터 (x₁, x₂, …, xd)
  • Y = {+1, -1}: 출력 공간 (이진 결정)
  • f: X → Y: 미지의 타겟 함수
  • N: 훈련 예시 개수
  • D = {(x₁,y₁), …, (xN,yN)}: 데이터셋 (yn = f(xn))

📊 신용 승인 데이터 시각화 (Credit Approval Scatter Plot)

x₁ = age (나이), x₂ = annual salary (연봉, k$)

○ (파란 원) = approve (승인) — 주로 연봉이 높은 영역

× (빨간 X) = deny (거절) — 연봉이 낮은 영역

g: 가설 함수 (학습된 결정 경계 직선)

f: 실제 타겟 함수 (알 수 없음, 근사 목표)

학습 알고리즘은 훈련 예시에서 g ≈ f를 만족하는 최적의 직선(결정 경계)을 탐색한다.

Problem Setup — Basic Setup & Learning Model

🔄 Basic Setup — 머신러닝의 기본 구조 (Slide 20)

🔧 Learning Model = 두 가지 도구

주어진 학습 문제에서 결정되는 것:

  • 타겟 함수 f: 문제에 의해 결정됨
  • 훈련 예시 D: 문제와 함께 제공됨

우리가 선택하는 것 (= Learning Model):

  • 가설 집합 ℋ: 후보 공식들의 집합
  • 학습 알고리즘 𝒜: ℋ에서 최적의 g 선택

🎯 Hypothesis Set ℋ

각 가설 h ∈ ℋ는 동일한 함수 형태 h(x)를 공유한다.

  • 각 좌표(feature)에 다른 가중치를 부여
  • 신용 결정에서의 상대적 중요도를 반영
  • 여기서의 선택: 선형 모델 (Linear Model)
  • ℋ = 직선들의 집합 (set of lines)

A Simple Learning Model — 의사결정 메커니즘

📊 Making a Decision — 가중합과 임계값

입력 x = (x₁, x₂, …, xd) (고객의 속성들)에 대해:

approve credit if Σᵢ wᵢxᵢ > threshold

deny credit if Σᵢ wᵢxᵢ < threshold
핵심: 각 feature에 가중치(weight) wᵢ를 곱해 합산한 점수를 임계값(threshold)과 비교한다.
bias 도입: bias b = −threshold 로 정의하면, 조건을 Σwᵢxᵢ + b > 0 으로 단순화할 수 있다.
05

Perceptron & PLA

⚡ Perceptron 공식

임계값을 bias b로 흡수하면:

h(x) = sign( (Σᵢ wᵢxᵢ) + b )

where sign(s) = +1 (s > 0), −1 (s < 0)

bias를 w₀ ≡ b, 인공 좌표 x₀ = 1 로 확장하면:

h(x) = sign(wTx)
  • w: 가중치 벡터
  • x: 입력 벡터 (x₀=1 포함)
  • 결정 경계: wTx = 0 (초평면)

🔁 PLA 알고리즘 (Perceptron Learning Algorithm)

목표: 훈련 데이터로부터 최적의 w를 찾아 g 생성

가정: 데이터가 선형 분리 가능 (linearly separable)

# PLA 의사코드 def PLA(D, w_init): w = w_init while misclassified_exists(D, w): # 잘못 분류된 점 (xn, yn) 선택 xn, yn = pick_misclassified(D, w) # sign(w^T xn) ≠ yn 인 점 # 가중치 업데이트 w = w + yn * xn # w(t+1) ← w(t) + yn * xn return w # final hypothesis g
수렴 보장: 데이터가 선형 분리 가능하면 PLA는 반드시 수렴한다. 선형 분리 불가능한 경우 → 수렴 불가.

🔄 PLA 가중치 업데이트 원리 (Weight Update Visualization)

y = +1 (approve) 인데 잘못 분류된 경우:

w(t+1) = w(t) + (+1)·x → w가 x 방향으로 이동 → 결정 경계가 x를 +1 영역으로 밀어냄

y = −1 (deny) 인데 잘못 분류된 경우:

w(t+1) = w(t) + (−1)·x → w가 x 반대 방향으로 이동 → 결정 경계가 x를 −1 영역으로 밀어냄

핵심 원리: 각 업데이트는 선택된 오분류 점을 올바르게 분류하는 방향으로 결정 경계를 이동시킨다.

🗺️ 퍼셉트론 결정 경계 — 2차원 케이스 (Slide 27)

(a) 오분류 데이터 — 경계 개선 필요

(b) 완벽 분류 — PLA 수렴

2차원 결정 경계: 가중치 벡터 w₁, w₂, bias b에 따라 직선 w₁x₁ + w₂x₂ + b = 0 이 결정된다. +1 결정 영역(파랑)과 −1 결정 영역(빨강)으로 평면을 분할.

Types of Learning — 학습 패러다임

⚖️ 세 가지 학습 패러다임 비교

구분 지도 학습 (Supervised) 강화 학습 (Reinforcement) 비지도 학습 (Unsupervised)
훈련 데이터 형태 (input, correct output) (input, some output, grade) (input only, no label)
정답 레이블 ✅ 모든 샘플에 존재 ⚠️ 없음 (보상/점수만) ❌ 없음
대표 알고리즘 Classification, Regression Q-Learning, PPO, DQN k-means, PCA, HMM
대표 예시 동전 인식, 스팸 필터 게임 플레이, 로봇 제어 동전 클러스터링, 토픽 분류
난이도 상대적으로 명확 탐색-활용 트레이드오프 클러스터 수 등 모호성

👩‍🏫 지도 학습 (Supervised)

가장 많이 연구되고 활용되는 학습 유형

  • 각 입력에 정확한 정답이 제공됨
  • 어떤 '지도자'가 각 입력을 보고 정답을 결정해 둔 것
  • 대표 접근: Classification, Regression
예: 동전 인식 자판기 — 크기/무게로 동전 종류 분류 (레이블 존재)

🎮 강화 학습 (Reinforcement)

정답이 없고, 행동에 대한 보상/점수만 존재

  • 마치 뜨거운 컵을 만진 아이가 경험으로 학습하듯
  • 훈련 예시: (입력, 어떤 출력, 그 출력의 점수)
  • 게임 학습에 특히 유용 (Chess, Go, Atari)
예: Tic-Tac-Toe — 지는 수에 페널티를 부여하며 전략 학습

🔍 비지도 학습 (Unsupervised)

출력 정보 없이 입력 데이터만 주어짐

  • Clustering: k-means, 혼합 모델, 계층적 클러스터링
  • Hidden Markov Models (HMM)
  • Feature extraction: PCA, ICA, SVD
  • 변형: Semi-supervised learning
예: 동전 클러스터링 — 레이블 없이 size/mass로 자동 그룹화

🎯 지도 학습의 핵심 구조 (Main Theme)

  • 미지의 타겟 함수 y = f(x) 존재
  • 알려진 훈련 데이터셋 (x₁,y₁), …, (xN,yN)
  • 학습 알고리즘이 가설 집합 ℋ에서 g ≈ f 선택
  • 새로운 입력 x에 대해 g(x)로 예측
Input space: X = ℝᵈ
Output space: Y = {+1, −1}
Target function: f: X → Y (unknown)
Training data: D = {(xₙ,yₙ)}ₙ₌₁ᴺ
Final hypothesis: g ≈ f

⚡ PLA 핵심 요약

  • 퍼셉트론: h(x) = sign(wᵀx)
  • 결정 경계: 2D에서 직선, dD에서 초평면
  • 업데이트: w(t+1) ← w(t) + yₙxₙ
  • 선형 분리 가능 → 수렴 보장
  • 선형 분리 불가능 → 수렴 보장 없음
  • +1 영역 (파랑): wᵀx > 0
  • −1 영역 (빨강): wᵀx < 0
  • 결정 경계: wᵀx = 0
  • 각 반복: 오분류 점 1개 선택 → 업데이트
  • 최종 g: 오분류 없을 때의 w
