Matplotlib 소개
What is Matplotlib?
  • Python의 포괄적인 시각화 라이브러리
  • 정적·애니메이션·인터랙티브 시각화 생성 가능
Why use Matplotlib?
  • 오픈소스 — 라이선스 비용 없음
  • 다양한 종류의 그래프 표현 지원
  • Python 스크립트, IPython 셸, Jupyter Notebook 호환
  • MATLAB과 유사한 함수 컬렉션 포함
기본 import: import matplotlib.pyplot as plt
함께 사용: import numpy as np

subplot() — 하나의 Figure에 여러 그래프
plt.subplot(nrows, ncols, index)
  • nrows: 행 수  |  ncols: 열 수  |  index: 위치 (1부터 시작)
  • plt.suptitle() 로 전체 Figure 제목 설정
# 1행 2열 (좌우) plt.subplot(1, 2, 1) plt.plot(x1, y1) plt.subplot(1, 2, 2) plt.plot(x2, y2) # 2행 1열 (위아래) plt.subplot(2, 1, 1) plt.plot(x1, y1) plt.subplot(2, 1, 2) plt.plot(x2, y2) plt.suptitle("MY SHOP")
subplot(1,2,1) / subplot(1,2,2) — 좌우
subplot(2,1,1) / subplot(2,1,2) + suptitle — 위아래

Seaborn 소개
What is Seaborn?
  • 통계 그래픽을 위한 시각화 라이브러리
  • Matplotlib 위에서 구축 (builds on top of)
  • Pandas 데이터 구조와 긴밀하게 통합
Why use Seaborn?
  • 데이터 탐색 및 이해를 도움
  • 함수가 전체 DataFrame/배열에서 직접 동작
  • 내부적으로 시맨틱 매핑과 통계 집계 자동 수행
기본 import: import seaborn as sns
테마: sns.set_theme(style="darkgrid")
seaborn.lineplot() 시그니처
seaborn.lineplot( data=None, *, x=None, y=None, hue=None, size=None, style=None, palette=None, markers=None, estimator='mean', errorbar=('ci', 95), n_boot=1000, ... )
data: pandas.DataFrame, numpy.ndarray, mapping or sequence

Seaborn 다양한 그래프
seaborn.lineplot() — semantic grouping 선 그래프
import seaborn as sns sns.set_theme(style="darkgrid") fmri = sns.load_dataset("fmri") sns.lineplot( x="timepoint", y="signal", hue="region", style="event", data=fmri )
  • hue: 색상으로 그룹 구분
  • style: 선 스타일로 그룹 구분
  • 자동으로 신뢰구간(CI) 음영 표시

seaborn.boxplot()
  • 카테고리별 분포를 박스플롯으로 표시
  • hue로 중첩 그룹핑

seaborn.violinplot()
  • 박스플롯 + KDE 결합

  • inner="quart": 사분위수 표시

seaborn.jointplot() / seaborn.kdeplot()
  • jointplot(): 두 변수의 이변량+단변량 그래프 동시 표시
  • kdeplot(): 커널 밀도 추정으로 분포 시각화
  • fill=True: 밀도 영역 채우기
  • hue: 그룹별 다른 색상

Summary — Matplotlib vs Seaborn
MATPLOTLIB
  • Low-level 라이브러리
  • 범용 플로팅
  • 플롯 요소에 대한 세밀한 제어
  • 복잡한 플롯에 코드가 많이 필요
  • 기본 기본값, 수동 스타일링 필요
  • Pandas와 통합되지만 매끄럽지 않음
  • 커스텀·복잡·저수준 플롯에 적합
SEABORN
  • High-level 라이브러리
  • 통계 데이터 시각화
  • 일반 작업에 간소화된 제어
  • 통계 플롯에 간결한 문법
  • 매력적인 기본값, 내장 테마
  • Pandas DataFrame과 완벽 통합
  • 빠르고 보기 좋은 통계 플롯에 적합
