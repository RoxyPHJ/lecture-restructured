전치행렬 (Transposed Matrix) Slide 8

행렬 A의 전치행렬 Aᵀ: 행과 열을 교환 — (A)ᵢⱼ → (Aᵀ)ⱼᵢ

A = [[3, 4, 1], Aᵀ = [[3, 0], [0, 5, 2]] [4, 5], [1, 2]]
m×n 행렬을 전치하면 n×m 행렬이 됨
행렬 덧셈 & Broadcasting Slide 9
  • 같은 shape의 행렬끼리 원소별(element-wise) 덧셈
  • C = A + B → Cᵢⱼ = Aᵢⱼ + Bᵢⱼ
  • 스칼라 곱: D = aB + c → Dᵢⱼ = aBᵢⱼ + c
Broadcasting: 벡터 b를 행렬 A의 각 행에 더하는 것
C = A + b → Cᵢⱼ = Aᵢⱼ + bⱼ
행렬 곱셈 Slide 10–11
  • A(m×n) × B(n×p) = C(m×p) — 내부 차원이 같아야 함
  • Cᵢⱼ = Σₖ Aᵢₖ Bₖⱼ
  • 원소별 곱(Hadamard product): A ⊙ B
주요 성질: 분배법칙 ✓ / 결합법칙 ✓
교환법칙 ✗ (AB ≠ BA) / 전치: (AB)ᵀ = BᵀAᵀ

역행렬 (Inverse Matrix) & 행렬식 Slide 16

정방행렬 A의 역행렬: A⁻¹A = Iₙ

Ax = b → x = A⁻¹b det([[a,b],[c,d]]) = ad - bc

행렬식은 변환이 공간을 얼마나 확장/축소/반전시키는지를 나타냄

■ 선형변환과 퍼셉트론 Slide 13–14

선형변환 Ax = b

행렬 A는 벡터 x를 벡터 b로 변환하는 선형 변환으로 해석 가능. n개의 미지수에 대한 n개의 연립방정식.

A₁₁x₁ + A₁₂x₂ + … + A₁ₙxₙ = b₁ A₂₁x₁ + A₂₂x₂ + … + A₂ₙxₙ = b₂ … Aₙ₁x₁ + Aₙ₂x₂ + … + Aₙₙxₙ = bₙ

퍼셉트론 f(x) = sign(wᵀx)

이진 분류 알고리즘. x는 입력 벡터, w는 가중치 벡터. 내적 wᵀx의 부호로 클래스 결정.

02
Vector & Matrix Norms
Slide 18–21
노름(Norm)의 정의 — 벡터의 크기를 측정

노름은 벡터를 비음수로 매핑하는 함수. 원점에서 벡터 x까지의 거리.

노름 함수 f 가 만족해야 할 조건: ① f(x) = 0 ⟹ x = 0 (영벡터만 크기 0) ② f(x + y) ≤ f(x) + f(y) (삼각 부등식) ③ ∀α ∈ ℝ: f(αx) = |α|f(x) (스칼라 상수배)
Lᵖ 노름 종류 비교
노름수식별명특징 & 응용
L² 노름 ||x||₂ = √(Σ xᵢ²) Euclidean norm 원점에서의 유클리드 거리 / Ridge Regression, Gradient Descent, 유클리드 거리
L¹ 노름 ||x||₁ = Σ|xᵢ| Manhattan norm 0과 비0을 구분할 때 유용 / Lasso Regression, Sparse feature selection — 작은 가중치를 0으로 만듦
L∞ 노름 ||x||∞ = maxᵢ|xᵢ| Max norm 가장 큰 원소의 절댓값 / 최악의 경우 측정 (worst-case scenario)
Frobenius ||A||_F = √(Σᵢⱼ Aᵢⱼ²) 행렬 L² 유사 행렬의 크기 측정 — 모든 원소를 제곱합 후 루트
■ Lᵖ 노름에 따른 단위원 형태 (p값 변화)
L² 활용: 선형회귀 & 정규화 Slide 20
y(x,w) = w₀ + w₁x₁ + … + wdyd = wᵀx E(w) = (1/2)Σᵢ{y(xₙ,w) - tₙ}² + (λ/2)||w||²
두 번째 항 = 정규화 항 (Regularizer) — 과적합 방지
L² 활용: 코사인 유사도 Slide 20
cosine_similarity(A,B) = cos(θ) = (A·B) / (||A||·||B||) θ → 0°: cos(θ) ≈ 1 (매우 유사) θ → 90°: cos(θ) ≈ 0 (무관) θ → 180°:cos(θ) ≈ -1 (반대 방향)

Special Kinds of Matrices & Vectors
Slide 23–26
대각행렬 (Diagonal Matrix) Slide 23–24
  • 대각선 원소만 0이 아닌 행렬
  • 예: 단위행렬(모든 대각원소 = 1)
  • 예: 독립 특징의 공분산 행렬
d(v)x = v ⊙ x (원소별 곱으로 단순화) d(v)⁻¹ = d(1/v₁, …, 1/vₙ)ᵀ (역행렬도 효율적)
v = (2,5,3) → d(v) = diag(2,5,3)
대칭행렬 (Symmetric Matrix) Slide 25
  • A = Aᵀ — 전치행렬이 자기 자신
  • 예: 거리행렬 (Aᵢⱼ = Aⱼᵢ)
  • 예: 공분산 행렬
공분산 행렬은 반드시 대칭. d(i,A)→d(j,A) 거리 = d(j,A)→d(i,A) 거리
단위벡터 / 직교벡터 / 직교정규벡터 / 직교행렬 Slide 26
개념조건의미
단위벡터 (Unit Vector) ||x||₂ = 1 L² 노름이 1인 벡터 — 방향만 표현
직교벡터 (Orthogonal) xᵀy = 0 내적이 0 = 두 벡터가 90° — 서로 수직
직교정규벡터 (Orthonormal) 직교 + 단위노름 xᵀy = 0 이고 ||x||₂ = ||y||₂ = 1
직교행렬 (Orthogonal Matrix) AᵀA = AAᵀ = I
Aᵀ = A⁻¹		 행 벡터들이 서로 직교정규 — 역행렬 = 전치행렬
04
Matrix Decomposition
Slide 28–38
행렬 분해의 동기 Slide 28

정수 인수분해처럼 행렬을 인수로 분해해 표현만으로는 보이지 않는 보편적 성질을 파악.

12 = 2 × 2 × 3 → 5로 나눌 수 없음 / 3의 배수임을 바로 알 수 있음
→ 행렬도 분해를 통해 고유한 구조적 정보 추출
■ 고유벡터 (Eigenvector) & 고유값 (Eigenvalue) Slide 29–30
Av = λv
  • 정방행렬 A에 곱했을 때 방향이 변하지 않고 크기만 변하는 비영 벡터 v
  • λ = 스케일 변화량 (고유값)
  • sv도 같은 고유값의 고유벡터 → 단위 고유벡터(||v||=1) 사용 관례

예시: A = [[2,1],[1,2]]

det(A - λI) = λ² - 4λ + 3 = 0 → λ₁ = 1, λ₂ = 3 v(λ=1) = (1,-1)ᵀ v(λ=3) = (1, 1)ᵀ
고유값 분해 (Eigendecomposition) Slide 31

행렬 A가 n개의 선형독립 고유벡터 {v₁,…,vₙ}와 고유값 {λ₁,…,λₙ}를 가질 때:

A = V · d(λ) · V⁻¹ V = [v₁ | v₂ | … | vₙ] (고유벡터를 열로 쌓은 행렬) d(λ) = diag(λ₁, λ₂, …, λₙ) (대각에 고유값 배치)
양정치 행렬 (Positive Definite, PD): 모든 고유값 > 0 → xᵀAx > 0
양반정치 (PSD): 모든 고유값 ≥ 0 → xᵀAx ≥ 0
■ 특이값 분해 (SVD — Singular Value Decomposition) Slide 34–37

고유값 분해는 정방행렬에만 적용 가능. SVD는 모든 실수 행렬에 적용 가능한 더 일반적인 분해.

A = U Σ Vᵀ U ∈ ℝᵐˣᵐ : 직교행렬 (AAᵀ의 고유벡터) Σ ∈ ℝᵐˣⁿ : 대각행렬 (특이값 σᵢ = √λᵢ) V ∈ ℝⁿˣⁿ : 직교행렬 (AᵀA의 고유벡터)
고유값 분해와의 연결:
• 좌특이벡터 = AAᵀ의 고유벡터
• 우특이벡터 = AᵀA의 고유벡터
• 특이값 σᵢ = AᵀA (또는 AAᵀ) 고유값의 제곱근
SVD 계산 예시 Slide 36–37
A = [[3, 6], AAᵀ = W₁ → 고유값: λ₁=57.884, λ₂=0.155, λ₃=0, λ₄=0 [2, 3], [0, 0]] AᵀA = W₂ → 고유값: λ₁=57.884, λ₂=0.155 특이값: σ₁ = √57.884 ≈ 7.605, σ₂ = √0.155 ≈ 0.394 U = eigenvectors(AAᵀ) = [[0.881, -0.471], [0.471, 0.881], [0,0], [0,0]] Vᵀ= eigenvectors(AᵀA) = [[0.471, -0.881], [0.881, 0.471]] Σ = diag(7.605, 0.394, 0) A = U · Σ · Vᵀ ✓
SVD 응용: 협업 필터링 (Collaborative Filtering) Slide 38

사용자-아이템 평점 행렬 R (m×n)을 SVD로 분해해 차원 축소 후 빠진 평점 예측.

R(m×n) = U(m×m) · Σ(m×n) · Vᵀ(n×n) ↓ k개의 대표 특이값만 사용 (차원 축소) R̂(m×n) ≈ Û(m×k) · Σ̂(k×k) · V̂ᵀ(k×n)
k개의 잠재 인수(latent factor)만 사용 → 노이즈 제거 + 계산 효율화

선형대수는 데이터를 구조화된 방식으로 저장·변환·분석하는 수학적 기반을 제공.

📊 벡터 — 데이터 표현
데이터 포인트나 특징을 벡터로 표현. 머신러닝의 기본 입력 단위.

여러 샘플 = 행렬. 행=샘플, 열=특징. 변환(회전·스케일)도 행렬.
⚙️ 행렬 연산
스케일링·회전·투영 등 데이터 전처리와 모델 계산의 핵심.
📐 연립방정식
Ax=b 형태 최적화 문제 해결. 역행렬·행렬식 활용.
🔬 SVD / 고유분해
차원 축소(PCA), 추천 시스템(CF), 노이즈 제거 등 핵심 알고리즘.

경사하강법·그래디언트는 벡터·행렬 미적분으로 표현.
