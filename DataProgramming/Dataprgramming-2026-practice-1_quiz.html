⭐⭐ 보통

💡 서술형은 채점에서 제외됩니다. 모범답안을 참고하세요.

모범답안
numpy를 사용하는 주요 이유는 다음과 같다.

① 벡터화(Vectorization): numpy는 C/Fortran으로 구현된 내부 루프를 사용하여 파이썬 for문 없이 배열 전체에 연산을 적용한다. 이로 인해 파이썬 리스트 기반 반복보다 수십~수백 배 빠른 속도를 낸다.

② 균일 자료형(Homogeneous dtype): ndarray의 모든 원소는 동일한 dtype을 가지므로, 메모리를 연속적으로 배치할 수 있어 캐시 효율이 높다.

③ 브로드캐스팅(Broadcasting): shape이 다른 배열 간 연산을 자동으로 확장 처리하여 코드를 간결하게 작성할 수 있다.

④ 다차원 배열 지원: 행렬, 3차원 이상의 텐서 연산을 직관적인 API로 처리 가능하다.

ES-02
아래 행렬 A와 B가 있을 때 AT×B를 구하라.
A = [[0.1, 3], [0.5, 2], [0.7, 3], [0.4, 1], [0.2, 2]] (5×2 행렬) / B = [3, 2, 1, 2, 0] (5×1 벡터 또는 5차원 열벡터로 해석)
⭐⭐⭐ 어려움
모범답안
A는 5×2 행렬, AT는 2×5 행렬.
B는 5×1 벡터. → AT×B는 (2×5)×(5×1) = 2×1 결과.

계산 과정:
AT = [[0.1, 0.5, 0.7, 0.4, 0.2],
        [3, 2, 3, 1, 2]]
B = [3, 2, 1, 2, 0]

결과[0] = 0.1×3 + 0.5×2 + 0.7×1 + 0.4×2 + 0.2×0
       = 0.3 + 1.0 + 0.7 + 0.8 + 0 = 2.8

결과[1] = 3×3 + 2×2 + 3×1 + 1×2 + 2×0
       = 9 + 4 + 3 + 2 + 0 = 18

∴ AT×B = [[2.8], [18]]

지도학습과 비지도학습의 차이점을 설명하고, 각각이 필요한 상황의 예시를 들어라.
⭐⭐ 보통
모범답안
지도학습(Supervised Learning): 입력 데이터와 정답 레이블(label)이 쌍으로 주어진 데이터로 모델을 학습. 모델은 입력→출력 매핑을 학습.
예시: 이메일 스팸 분류(스팸/정상 레이블 존재), 이미지 분류(고양이/개 레이블 존재)

비지도학습(Unsupervised Learning): 정답 레이블 없이 데이터의 내재적 구조(패턴, 분포)를 스스로 학습.
예시: 고객 구매 패턴 군집화(레이블 없이 비슷한 고객끼리 묶기), 이상 탐지(정상 패턴에서 벗어난 데이터 식별)

응용pandas DataFrame과 numpy ndarray의 차이점을 3가지 관점에서 비교 설명하라. (자료형, 인덱스, 활용 목적)
⭐⭐ 보통
모범답안
① 자료형(dtype): ndarray는 모든 원소가 동일한 dtype(homogeneous). DataFrame은 열(column)마다 다른 dtype을 허용(heterogeneous) — 예: int 열과 str 열 공존 가능.

② 인덱스: ndarray는 정수 위치 인덱스만 사용. DataFrame은 사용자 정의 레이블 인덱스(행 인덱스 + 열 이름)를 지원하여 의미있는 접근이 가능.

③ 활용 목적: ndarray는 수치 연산, 선형대수, 과학계산에 최적화. DataFrame은 테이블형 데이터 분석(결측치 처리, 그룹화, 조인 등)에 특화.
