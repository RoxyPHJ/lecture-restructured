01

Regression 개요

Regression이란?

x와 y 사이의 관계를 통계적으로 탐구하는 방법론

  • x: covariate / predictor variable / independent variable / feature
  • y: response / dependent variable
  • 노이즈 포함: yₙ = f(xₙ) + ε
  • 목표: y ~ P(y|x) 분포에서 yₙ을 근사하는 모델 g(x) 찾기
Training Data
(x₁, y₁), (x₂, y₂), …, (xₙ, yₙ)

어원: Francis Galton (1822–1911)

  • re (back) + gression (going)
  • 데이터에서 공식으로 역추적하는 행위
  • Regression towards the mean
    "키 큰/작은 아버지의 아들은 평균에 가까워진다"
y = f(x) 결정론적 함수가 아니라
y ~ P(y|x) 확률적 타겟으로 접근

Simple Linear Regression 시각화

각 xₙ에서 y는 결정론적 값이 아니라 분포 P(y|x)에서 샘플링됨. 빨간 선 f가 참 함수, 파란 점이 관측값 yₙ = f(xₙ) + ε

02

Linear Regression이란

정의 & 특징

  • 정량적 반응 예측을 위한 인기 있는 선형 모델
  • 실수값(real-valued) 타겟 함수에 적용
  • 통계학, 사회과학 분야에서 오랜 역사
  • "regression" ⇒ y가 실수값임을 의미 (통계학에서 계승)

Simple vs Multiple

구분설명
d = 1Simple Linear Regression (예측변수 1개)
d ≥ 2Multiple Linear Regression (예측변수 다수)

강의에서는 학습 관점에서 simple linear regression을 중심으로 논의

Linear Regression in 1D and 2D

(a) 1D — Line

(b) 2D — Hyperplane

OLS에서는 잔차(residual)의 제곱합을 최소화하는 파란색 직선/평면을 찾는다.

05

Error 함수 정의

Out-of-sample Error (E_out)

정의 (슬라이드 15)
E_out(h) = 𝔼[(h(x) − y)²]
  • h(x)와 y의 제곱 오차의 기댓값
  • 문제점: P(x,y)를 모르므로 계산 불가능

In-sample Error (E_in)

정의 (슬라이드 16)
E_in(h) = (1/N) Σ(h(xₙ) − yₙ)²
  • 훈련 데이터에 대한 평균 제곱 오차
  • 계산 가능: 실제 최적화 대상
  • h(x) = Σᵢ wᵢxᵢ = wᵀx (선형 조합)

핵심 접근 전략

P(x,y)를 모르므로 E_out 계산이 불가능 → 대신 E_in을 최소화하는 w를 찾고, 이 w가 E_out도 작기를 기대

Hypothesis 형태 (Signal)
h(x) = w₀x₀ + w₁x₁ + … + wdxd = wᵀx    (x₀ = 1, bias term)
06

행렬 표현 (Matrix Representation)

데이터 행렬과 타겟 벡터

X ∈ ℝ^(N×(d+1)) — 데이터 행렬

  • 행: 입력 xₙ (행 벡터로)
  • 첫 열: bias term (모두 1)
d=1, N=4 예시
X = [[1, x₁],
     [1, x₂],
     [1, x₃],
     [1, x₄]]

y ∈ ℝ^N — 타겟 벡터

  • 성분: 타겟값 yₙ
w, y 벡터
w = [w₀, w₁]ᵀ

y = [y₁, y₂, y₃, y₄]ᵀ

행렬 형태의 E_in

전개 과정 (슬라이드 18)
E_in(w) = (1/N) ||Xw − y||²

           = (1/N)(wᵀXᵀXw − 2wᵀXᵀy + yᵀy)

주의: scalar이므로 yᵀXw = wᵀXᵀy가 성립 (전치 트릭 사용)

07

E_in 최소화 (Minimization)

E_in의 특성

  • 연속 (Continuous)
  • 미분 가능 (Differentiable)
  • 볼록 (Convex)

→ 기울기=0 조건으로 전역 최솟값 직접 도출 가능

Gradient Identity (행렬 미적분)

벡터 미분 공식 (슬라이드 22)
∇_w (wᵀAw) = (A + Aᵀ)w
∇_w (wᵀb) = b
E_in 그래디언트
∇E_in(w) = (2/N)(XᵀXw − Xᵀy)

최적화 조건

∇E_in(w) = 0
⇒ XᵀXw = Xᵀy

이것이 정규방정식 (Normal Equation)

E_in 볼록 함수 (Convex Bowl)

볼록 함수이므로 local minima = global minimum. 등고선의 중심이 w_lin

08

풀이 & 선형 회귀 알고리즘

두 가지 시나리오

Case 1: XᵀX가 역행렬 존재 (가역)

w = X†y = (XᵀX)⁻¹Xᵀy
X† : X의 pseudo-inverse
→ 유일한 최적해 w_lin

Case 2: XᵀX가 singular (비가역)

  • pseudo-inverse 정의 가능하나 해 비유일
  • E_in을 최소화하는 w가 여럿 존재

실제로는: N ≫ d+1이면 대부분 가역 → Case 1이 일반적

Pseudo-inverse via SVD

SVD 분해: X = UΣVᵀ
U ∈ ℝ^(N×ρ), UᵀU = I_ρ
V ∈ ℝ^((d+1)×ρ), VᵀV = I_ρ
Σ ∈ ℝ^(ρ×ρ) 양의 대각 행렬
ρ = rank(X)

w_lin = VΣ⁻¹Uᵀy
  • 행렬 역산 없이 X†를 직접 계산
  • 수치 안정성 더 우수

선형 회귀 알고리즘 (3-Step)

1

데이터셋 {(x₁,y₁), …, (xₙ,yₙ)}으로 데이터 행렬 X와 타겟 벡터 y 구성

각 x에 bias 좌표 x₀ = 1 포함

2

X의 pseudo-inverse X† 계산

XᵀX가 가역이면: X† = (XᵀX)⁻¹Xᵀ
그렇지 않으면: 수치 안정적인 구현 사용
3

w_lin = X†y 반환

→ "1-step learning": 반복 없이 분석 해 직접 계산

PLA와 비교: 선형 회귀는 학습인가?

항목PLA (분류)Linear Regression
학습 방식반복적 업데이트분석적 해 (1-step)
타겟이진 {+1, -1}실수 ℝ
오차 함수오분류 개수제곱 오차
해 형태w = iterativew = (XᵀX)⁻¹Xᵀy

→ 1-step이지만 학습이 발생함. E_out이 작다면 일반화 성공
