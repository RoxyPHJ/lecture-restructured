01 소개 & 동기
선형 모델의 핵심 아이디어

모든 선형 모델은 "신호" s를 공유한다. 입력 변수들을 선형 결합해 단일 스칼라를 만든다.

s = Σᵢ wᵢxᵢ = wᵀx
① 선형 회귀
  • 출력: 신호 s 자체 → h(x) = wᵀx
  • 제한 없는 실수 예측 (연속값)
② 선형 분류
  • 신호를 0에서 임계값 처리 → h(x) = sign(wᵀx)
  • ±1 이진 결정
동기: 심장마비 예측

콜레스테롤, 혈압, 나이, 체중 등 입력으로 심장마비를 예측할 때, 확실하게 예측하기는 불가능하다. 대신 발생 가능성(확률)을 예측하는 것이 더 적합하다.

  • 출력 y가 0~1 사이로 연속적으로 변해야 함
  • y가 1에 가까울수록 심장마비 가능성 높음
  • → "soft label" (확률) 반환 모델 필요
로지스틱 회귀의 핵심

출력이 회귀처럼 실수이면서 분류처럼 유계(bounded)
→ 이진 사건에 대한 불확실성을 허용하는 "soft binary classification"

02 로지스틱 함수 θ (Sigmoid)
정의
θ(s) = eˢ / (1 + eˢ) = 1 / (1 + e⁻ˢ)
  • 입력: -∞ < s < ∞ (모든 실수)
  • 출력: 0 과 1 사이 → 확률로 해석 가능
  • "soft threshold": 하드 분류의 부드러운 버전
  • "sigmoid": 납작해진 S 모양
1 - θ(s) = θ(-s) [대칭성]
이 공식을 사용하는 이유: 오류 측도를 분석적·계산적으로 편리하게 정의할 수 있기 때문
tanh (또 다른 시그모이드)
tanh(s) = (eˢ - e⁻ˢ) / (eˢ + e⁻ˢ)
  • |s| 클 때 → 하드 임계값으로 수렴
  • |s| 작을 때 → 임계값 없음 (선형)
03 세 가지 선형 모델 비교
심장마비 예측 예시로 본 세 모델
모델 h(x) 출력 의미 활성함수
선형 분류 ±1 심장마비 발생(+1) / 미발생(-1) 확정 판단 sign(wᵀx)
선형 회귀 s ∈ ℝ 위험 점수 자체 (범위 무제한) 없음 (identity)
로지스틱 회귀 θ(s) ∈ [0,1] 심장마비 확률 (불확실성 허용) θ(wᵀx) [sigmoid]
신호 → 출력 변환 아키텍처
04 확률 예측 (Predicting Probability)
학습 목표 (Learning Target)
f(x) = ℙ[y = +1 | x]

입력 x가 주어졌을 때 y=+1일 확률. 이것이 로지스틱 회귀가 근사하려는 진짜 함수.

이상적 데이터 (확률값)
(x₁, f₁=0.99 = ℙ[+1|x₁])
(x₂, f₂=0.13 = ℙ[+1|x₂])
⋮
(xN, fN=0.51 = ℙ[+1|xN])
실제 데이터 (이진 사건)
(x₁, y₁ = +1)
(x₂, y₂ = -1)
⋮
(xN, yN = -1)

훈련 데이터는 f를 직접 주지 않는다. 대신 그 확률로 생성된 샘플을 준다.

Big Picture: 모델의 동작 방식

로지스틱 회귀는 h(x) = θ(wᵀx)로 ℙ[y=+1|x]를 근사한다. 이 h를 이용해 각 데이터포인트에 대한 확률을 통합 공식으로 표현할 수 있다.

ℙ[y=+1|x] ≈ h(x) = θ(wᵀx)
P(y|x) = h(x)^[y=+1] · (1-h(x))^[y=-1] = θ(y·wᵀx)

y=+1이면 h(x), y=-1이면 1-h(x) = θ(-wᵀx). θ의 대칭성(1-θ(s)=θ(-s)) 덕분에 θ(y·wᵀx) 로 통합된다.

05 교차 엔트로피 오류 (Cross-Entropy Error)
최대 우도법 (Maximum Likelihood)

샘플들이 독립적으로 생성되었다고 가정하면, 전체 훈련 데이터의 우도(likelihood)는:

Π_{n=1}^{N} P(yₙ|xₙ) = Π_{n=1}^{N} θ(yₙwᵀxₙ)

최대 우도법: 이 확률을 최대화하는 h를 선택한다. 등가적으로 -(1/N)ln(⋅)을 최소화한다.

E_in(w) = (1/N) Σ ln(1 + e^{-yₙwᵀxₙ}) ← Cross-Entropy Error
크로스 엔트로피 유도

이진 PMF {p, 1-p}와 {q, 1-q}에 대한 교차 엔트로피:

p·log(1/q) + (1-p)·log(1/(1-q))

관측: {[yₙ=+1], 1-[yₙ=+1]}
피팅: {h(xₙ), 1-h(xₙ)}
→ 이 두 PMF 간의 교차 엔트로피 = NLL의 각 항

핵심 등치 관계

E_in(w) 최소화
= NLL 최소화
= Likelihood 최대화

E_in(w) = (1/N) Σ ln(1 + e^{-yₙwᵀxₙ})
Cross-Entropy의 장점: 볼록 함수(Convex)

교차 엔트로피 E_in(w)은 w에 대해 볼록 함수(convex function)이다.

  • Hessian이 양반정치(positive semidefinite)
  • 극솟값이 하나만 존재 (유일한 전역 최솟값)
  • 시작점에 무관하게 경사 하강법이 지역 최솟값에 갇히지 않음
06 경사 하강법 (Gradient Descent)
왜 경사 하강법인가?
모델 최적화 방법
선형 분류 조합적 최적화 (PLA/pocket)
선형 회귀 ∇E_in=0 해석적 해 (pseudo-inverse)
로지스틱 회귀 ∇E_in=0 → 해석적 해 없음!
→ 반복 최적화 (경사 하강법)
∇E_in(w) (로지스틱)
∇E_in(w) = -(1/N) Σ yₙxₙ·θ(-yₙwᵀxₙ)

비선형 방정식 → 해석적으로 0으로 놓고 풀기 불가

경사 하강법 원리

E_in(w)는 고차원 공간의 "표면". 공이 언덕에서 굴러내려오는 것처럼 가장 가파른 방향으로 이동.

  • 방향: 음의 그래디언트 방향 (가장 가파른 하강)
  • 크기: 학습률 η로 제어
Δw = -∇E_in(w)
w(t+1) = w(t) - η·∇E_in(w(t))
⚠ 주의: 지역 최솟값

일반적인 오류 함수에서는 시작 위치에 따라 지역 최솟값에 수렴할 수 있다. 하지만 교차 엔트로피는 볼록 함수라 이 문제가 없다.

학습률 η의 영향

η 너무 작음: 최솟값에서 멀 때 비효율적

η 너무 큼: 최솟값 근처에서 튀어다님, 발산 가능

이상적 전략: 최솟값에서 멀 때 큰 보폭 → 가까울 때 작은 보폭.
그래디언트 norm이 멀 때 크고 가까울 때 작음을 이용: η_t = η·‖∇E_in(w(t))‖ → 정규화 항이 자동으로 상쇄되어 w(t+1) = w(t) - η·∇E_in(w(t))

07 로지스틱 회귀 알고리즘 & 종료 기준
로지스틱 회귀 알고리즘 (최종)
1:Initialize w(0) ← 0 또는 작은 난수
2:for t = 0, 1, 2, ... do
3:// 그래디언트 계산
4:∇E_in(w(t)) = -(1/N) Σ yₙxₙ·θ(-yₙwᵀxₙ)
5:// 이동 방향
6:v_t = -∇E_in(w(t))
7:// 가중치 업데이트
8:w(t+1) = w(t) + η·v_t
9:until stopping criterion
10:return w(t)
w(t+1) = w(t) - η·∇E_in(w(t))
초기화 (Initialization)
  • w(0) = 0: 로지스틱 회귀에선 잘 동작
  • 난수 초기화: 일반적으로 더 안전
  • 대칭적 언덕꼭대기에 갇히는 것 방지
  • 실전: N(0, 작은 분산)에서 독립적으로 추출
종료 기준 (Stopping Criterion)
  • ① 최대 반복 횟수 설정 (상한)
  • ② 그래디언트 크기 임계값 이하
  • ③ 오류 변화량 작고 오류 자체도 작을 때
  • ⚠ 평탄 구간(plateau)에서 조기 종료 주의
  • 실전: ①+③ 조합 사용
경사 하강법 변형 (Variants)
방법 업데이트 당 샘플 수 장점 단점
Batch GD N개 전체 안정적 수렴, 정확한 그래디언트 대규모 데이터에 느림
Stochastic GD (SGD) 1개 빠른 업데이트, 지역 최솟값 탈출 가능 노이즈 많은 수렴
Mini-batch GD b개 (보통 2~100) 배치와 SGD의 균형, GPU 활용 효율적 batch size 선택 필요
08 세 선형 모델 최종 비교 & 요약
세 선형 모델 완전 비교표
선형 분류 (Perceptron) 선형 회귀 로지스틱 회귀
출력 공간 Y {-1, +1} ℝ (실수 전체) {-1, +1} 레이블 / 확률 [0,1]
가설 h(x) sign(wᵀx) wᵀx θ(wᵀx) ∈ [0,1]
오류 함수 e(ŷ,y) 0-1 손실: [ŷ≠y] 제곱 오류: (ŷ-y)² 교차 엔트로피: ln(1+e^{-ywᵀx})

📌 핵심 요약

1
로지스틱 회귀 = "soft" 이진 분류
출력 h(x) = θ(wᵀx) ∈ [0,1]으로 이진 사건의 확률을 추정. 선형 분류와 달리 불확실성을 허용하는 "soft label" 반환.
2
로지스틱 함수 θ(s) = 1/(1+e⁻ˢ)
"sigmoid" 또는 "soft threshold". 1-θ(s)=θ(-s) 대칭성 덕분에 P(y|x)=θ(y·wᵀx)로 통합 표현 가능.
3
교차 엔트로피 오류 = 최대 우도법의 등가
E_in(w) = (1/N)Σln(1+e^{-yₙwᵀxₙ}) 최소화 = NLL 최소화 = Likelihood 최대화. 볼록 함수이므로 유일한 전역 최솟값 보장.
4
경사 하강법: w(t+1) = w(t) - η·∇E_in(w(t))
해석적 해가 없으므로 반복 최적화 사용. 학습률 η는 약 0.1이 경험적으로 좋음. 볼록성으로 어디서 시작해도 전역 최솟값 도달.
5
선형 모델을 항상 먼저 시도하라
세 선형 모델은 동일한 선형 가설 집합을 공유하며, 단순하고 강건하고 잘 일반화된다. 데이터에서 학습할 때 첫 번째 시도는 항상 선형 모델.
