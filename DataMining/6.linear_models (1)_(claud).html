01

선형 모델 개요

The Linear Model
  • 직선들의 집합(set of lines)으로 이루어진 모델
  • 많은 경우 좋은 첫 번째 선택(good first choice)
  • 작은 VC 차원 → 학습 데이터(E_in)에서 테스트 데이터(E_out)로 잘 일반화
세 가지 핵심 문제 (슬라이드 3)
1. Classification

입력 데이터를 이산 클래스로 분류

2. Regression

연속적인 실수 값을 예측

3. Probability Estimation (Logistic Regression)

클래스에 속할 확률을 추정

02

이진 분류 (Binary Classification)

선형 분류기의 가설 (슬라이드 4)
  • 가설 집합: 선형 분류기들의 집합
  • w ∈ ℝd+1 : 가중치 열벡터 (weight column vector)
  • d : 입력 차원 (input dimension)
  • x₀ = 1 : 편향(bias) w₀에 대응
  • h와 w는 가설을 지칭하는 데 혼용
h(x) = sign(wTx)

※ PLA(Perceptron Learning Algorithm)는 선형 분리 가능한 데이터에서 최적 w를 찾음

선형 분리 불가 데이터의 두 유형 (슬라이드 5)
  • (a) 노이즈 포인트
    단 2개 예제를 이동하면 선형 분리 가능 → 노이즈/이상치
  • (b) 비선형 분리
    직선이 아닌 원(circle)으로만 분리 가능
상황 (a)
선형 모델 유지 + 노이즈 허용 (E_in을 0이 아닌 작은 값으로)
상황 (b)
비선형 변환(nonlinear transformation) 기법 필요
03

선형 분리 불가 & Pocket Algorithm

E_in 최소화의 어려움 (슬라이드 7)
  • 최소 E_in을 찾으려면 조합 최적화(combinatorial optimization) 필요
  • sign(·) 함수와 [·] 모두 이산적(discrete) → 일반적으로 NP-hard
  • 효율적인 알고리즘이 알려져 있지 않음
minw∈ℝd+1 (1/N) · Σ [sign(wTx) ≠ y]

→ 따라서 E_in을 근사적으로 최소화하는 접근이 필요

Pocket Algorithm (슬라이드 8~9)
  • PLA의 단순 수정으로 확장 (pocket algorithm with ratchet)
  • 매 반복마다 "주머니(pocket)"에 최선의 w를 보관
  • 최종적으로 가장 낮은 E_in을 가진 w를 반환
  • 단점: 각 반복에서 E_in 평가 필요 → PLA보다 훨씬 느림
1:set pocket weight vector ŵ ← w(0) of PLA
2:for t = 0, …, T−1 do
3:run PLA for one update → obtain w(t+1)
4:evaluate E_in(w(t+1)) /* time consuming */
5:if w(t+1) is better than ŵ in terms of E_in
6:    then set ŵ ← w(t+1)
7:return ŵ
PLA vs Pocket: 오류 비교 (슬라이드 14)
PLAPocket
E_in 2.24% 0.45%
E_out 6.37% 1.89%
반복 횟수 1000 (강제 종료) 1000
수렴 형태 불안정 (진동) 단조 감소

Pocket: E_in이 단조적으로 감소하며 안정화

회귀 (Regression) 개요

회귀란? (슬라이드 17~19)
  • x와 y 사이의 관계를 연구하는 통계적 방법
  • x: 공변량(covariate) / 예측 변수 / 독립 변수 / 특징
  • y: 응답(response) / 종속 변수
  • 목표에 노이즈 추가: y = f(x) + ε → y ~ P(y|x)
  • 목표: y를 근사하는 모델 h(x) 찾기
y = f(x) + ε → y ~ P(y | x)
어원 & 역사 (슬라이드 19)
  • Sir Francis Galton (1822~1911)이 처음 사용
  • re(back) + gression(going): 데이터에서 공식으로 돌아가기
  • "Regression towards the mean": 키 큰/작은 부모의 자녀 키는 평균에 가까워지는 경향
단순 vs 다중 선형 회귀 (슬라이드 20)
단순 선형 회귀 (d=1)

예측 변수 1개
2D 평면에서의 직선

다중 선형 회귀 (d≥2)

예측 변수 다수
고차원 초평면(hyperplane)

파라미터 추정 방법들 (슬라이드 21)
  • OLS (Ordinary Least Squares): 잔차 제곱합 최소화, 닫힌 형태(closed-form) 해 존재
  • Generalized / Iteratively Reweighted Least Squares
  • Maximum Likelihood
  • Ridge (Tikhonov regularization) / Lasso regression
  • Least Absolute Deviation regression
  • Bayesian linear regression, Principal component regression
07

선형 회귀 알고리즘

오류 함수 정의 (슬라이드 27~28)
  • E_out: h(x)와 y의 제곱 오차의 기댓값 (P(x,y)가 미지 → 계산 불가)
  • E_in: 학습 데이터 기반 샘플 평균 오차 (계산 가능)
E_out(h) = 𝔼[(h(x) − y)²]
E_in(h) = (1/N) · Σᵢ (h(xᵢ) − yᵢ)²

선형 회귀의 가설: h(x) = wTx (wTx = "signal")

신용 승인 예제 → 회귀로 (슬라이드 23~26)
구성요소기호신용 승인 비유
입력x고객 신청서
출력y승인/거절 또는 신용 한도액
목표 함수f: X → Y이상적인 신용 승인 공식
데이터(x₁,y₁),...,(xₙ,yₙ)과거 기록
가설h: X → Y사용할 공식

다수의 전문가가 레이블링 → 목표가 확정적 함수가 아닌 분포 P(y|x)로 표현됨

E_in(w)의 볼록성 (슬라이드 33)
  • E_in(w)는 연속, 미분 가능, 볼록(convex)
  • ∇E_in(w) = 0 으로 최솟값 찾기 가능
  • 일반 최적화 기법 사용 가능 (e.g. 경사 하강법)
  • 유일한 전역 최솟값 보장
E_in(w) = (1/N)(wTXTXw − 2wTXTy + yTy)
∇E_in(w) = 0
08

행렬 표현 & 해 도출

행렬 표현 (슬라이드 29~31)

데이터 행렬 X ∈ ℝN×(d+1)

X = [ 1 x₁ ] (d=1, N=4 예시)
[ 1 x₂ ]
[ 1 x₃ ]
[ 1 x₄ ]

목표 벡터 y ∈ ℝN

w = [ w₀ ] y = [ y₁ ]
[ w₁ ] [ y₂ ]
[ y₃ ]
[ y₄ ]
E_in(w) = (1/N) ‖Xw − y‖²
해 도출 과정 (슬라이드 34~36)
  • 그래디언트 항등식 활용:
∇E_in(w) = (2/N)(XTXw − XTy) = 0
XTXw = XTy
  • XTX가 역행렬 존재 시: 유일한 최적해
w_lin = X† y = (XTX)⁻¹XTy
  • X† = (XTX)⁻¹XT : X의 유사역행렬(pseudo-inverse)
  • XTX가 역행렬 없을 때: 유사역행렬은 여전히 정의 가능, 단 해가 유일하지 않음
선형 회귀 알고리즘 요약 (슬라이드 37~38)
1.데이터셋 (x₁,y₁), …, (xₙ,yₙ)으로 행렬 X와 벡터 y 구성
   (각 xᵢ에 편향 좌표 x₀=1 포함)
2.X의 유사역행렬 X† 계산 /* XTX 가역이면: X† = (XTX)⁻¹XT */
3.return w_lin = X†y

실제로 N ≫ d+1인 경우가 많아 XTX는 대부분 가역 → 많은 수학 라이브러리가 수치적으로 안정적인 X† 구현을 제공

핵심 요약

선형 모델의 세 가지 문제

선형 모델은 분류(Classification), 회귀(Regression), 확률 추정(Logistic Regression) 세 가지 주요 문제에 적용된다. 각각 다르지만 관련된 알고리즘을 사용한다.

이진 분류 & Pocket Algorithm

선형 분리 불가 데이터에서 최소 E_in을 찾는 것은 NP-hard 문제이다. Pocket Algorithm은 PLA를 단순 수정하여 최선의 w를 "주머니"에 보관하며 근사적으로 E_in을 최소화한다. 결과: E_in 0.45%, E_out 1.89% (PLA 대비 큰 개선).

선형 회귀 & OLS

선형 회귀는 h(x) = wTx 형태의 가설로 제곱 오차를 최소화한다. E_in(w)가 볼록 함수이므로 ∇E_in = 0 조건으로 단 한 번의 계산으로 전역 최적해를 구할 수 있다.

핵심 공식 & 알고리즘

w_lin = X†y = (XTX)⁻¹XTy

① X, y 행렬 구성 → ② 유사역행렬 X† 계산 → ③ w_lin = X†y 반환

산소/탄화수소 예제: g(x) = 74.28 + 14.95x, E_in = 1.06, E_out ≈ 1.45
