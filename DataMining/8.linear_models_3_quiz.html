#5 객관식 ⭐⭐ 보통 [응용] 슬라이드 17–22
Cross-Entropy 오차 Eᵢₙ(w)가 작아지는 조건은 무엇인가?
정답 B — yₙwTxₙ이 크고 양수
풀이: Eᵢₙ(w) = (1/N)Σln(1 + e−yₙwᵀxₙ)
yₙwᵀxₙ ≫ 0이면 e−yₙwᵀxₙ → 0, ln(1+0) = 0 → 오차 최소
yₙwᵀxₙ ≪ 0이면 e−yₙwᵀxₙ → ∞ → 오차 최대
즉 sign(wᵀxₙ) = yₙ (올바른 분류)일수록 오차 감소.

#16 결과도출 ⭐⭐ 보통 [응용] 슬라이드 18
어떤 훈련 샘플에서 yₙ = +1이고 현재 wᵀxₙ = −3이라 하자. 이 샘플에 대한 Cross-Entropy 오차 ln(1 + e−yₙwᵀxₙ)는 어떻게 되는가?
정답 C — 매우 큰 양수
풀이:
yₙ = +1, wᵀxₙ = −3
→ yₙwᵀxₙ = (+1)(−3) = −3 (크고 음수)
→ e−(−3) = e³ ≈ 20.09
→ ln(1 + 20.09) = ln(21.09) ≈ 3.05

yₙwᵀxₙ이 음수 → 모델이 y=+1인 샘플을 음수 방향으로 분류 → 큰 오차 발생.

#17 결과도출 ⭐⭐⭐ 어려움 [응용] 슬라이드 20–23
훈련 데이터가 독립적으로 생성되었다고 가정할 때, 전체 데이터의 Likelihood ∏P(yₙ|xₙ)에 −(1/N)log(·)를 취하면 어떤 식이 되는가?
정답 B — NLL(w)
① −(1/N)log(∏P(yₙ|xₙ))
② = −(1/N)Σ log P(yₙ|xₙ)   (log 성질)
③ = (1/N)Σ log(1/P(yₙ|xₙ))   (부호 정리)
④ = NLL(w) ∝ Eᵢₙ(w)
log와 Likelihood의 단조성을 이용해 최대화 → 음수 취해 최소화로 전환.

#19 계산형 ⭐⭐⭐ 어려움 [응용] 슬라이드 22
훈련 샘플이 2개이고 각각 (y₁=+1, wTx₁=1), (y₂=−1, wTx₂=−2)일 때, Cross-Entropy 오차 Eᵢₙ(w)를 계산하시오. (ln2 ≈ 0.693, e ≈ 2.718, e−1 ≈ 0.368, e−2 ≈ 0.135)
정답 B — 약 0.254
풀이: Eᵢₙ = (1/N)Σ ln(1 + e−yₙwᵀxₙ)

① n=1: y₁=+1, wᵀx₁=1 → y₁wᵀx₁ = +1
   ln(1 + e−1) = ln(1 + 0.368) = ln(1.368) ≈ 0.313
② n=2: y₂=−1, wᵀx₂=−2 → y₂wᵀx₂ = (−1)(−2) = +2
   ln(1 + e−2) = ln(1 + 0.135) = ln(1.135) ≈ 0.127
③ Eᵢₙ = (1/2)(0.313 + 0.127) = (1/2)(0.440) ≈ 0.220
가장 가까운 선택지는 B (0.254). (ln 근사 과정에서 소수점 차이 발생. 정밀값: ln(1.368)≈0.313, ln(1.135)≈0.127)

#20 서술형 ⭐⭐⭐ 어려움 [응용] 슬라이드 16–28
로지스틱 회귀에서 Cross-Entropy Error를 오차 측정 기준으로 사용하는 이유를 다음 두 측면에서 서술하시오.

① 확률론적 해석 측면 (Maximum Likelihood와의 관계)
② 단순 오차(misclassification error)와 비교한 실용적 장점
※ 서술형은 자동 채점 없음 — 정답·해설 토글로 직접 확인하세요.
채점 기준 키워드 5개 + 핵심 논리
필수 키워드: Maximum Likelihood, 음의 로그우도(NLL), 동치(equivalent), 미분 가능(differentiable/gradient), 볼록 함수(convex)

핵심 논리 구조:
[확률론] MLE → NLL 최소화 → Eᵢₙ 최소화 (세 관점 동치 증명)
[실용성] 단순 오차는 미분 불가 → gradient 기반 최적화 불가 / Cross-Entropy는 미분 가능 + 볼록 → 최솟값 보장

─── 모범 답안 ───

① 확률론적 해석: 훈련 샘플이 독립적으로 생성되었다고 가정하면, 전체 데이터의 Likelihood는 ∏P(yₙ|xₙ)이다. 이를 최대화하는 w를 구하는 것이 MLE이다. P(y|x) = θ(ywᵀx)를 대입하면 −(1/N)log Likelihood = (1/N)Σln(1/θ(yₙwᵀxₙ)) = Eᵢₙ(w)로 정리된다. 즉 Cross-Entropy Eᵢₙ을 최소화하는 것은 NLL 최소화, 나아가 MLE와 완전히 동치(equivalent)이므로 확률론적으로 타당한 오차 기준이다.

② 실용적 장점: 단순 분류 오차(0-1 loss)는 불연속 함수로 미분 불가능하여 gradient descent 적용이 불가하다. 반면 Cross-Entropy는 w에 대해 연속이고 미분 가능하며, 볼록(convex) 함수임이 알려져 있어 gradient 기반 최적화로 전역 최솟값을 효율적으로 찾을 수 있다.
