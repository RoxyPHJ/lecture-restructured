01

불확실성과 데이터 마이닝

Slide 2
핵심 개념
  • 데이터 마이닝의 많은 작업은 불확실성 하에서 예측을 수행
  • 분류(Classification): 입력 특징 벡터 X로부터 클래스 레이블 Y를 예측
  • 예: 이메일을 스팸/햄으로 분류 → 사후 확률 P(Y|X) 로 표현
  • 현실 데이터에는 다양한 원인으로 불확실성이 존재
  • 데이터 수집 오류로 인한 노이즈 데이터
  • 누락되거나 불완전한 속성
  • 클래스 간 특징 분포의 중첩
  • 측정 오류 및 환경적 변동성
  • 정보 이론은 확률 분포에서 불확실성을 수량화하는 수학적 프레임워크를 제공
  • 엔트로피, 조건부 엔트로피, 상호 정보 등의 개념 포함
💡
핵심 질문: 데이터의 불확실성을 어떻게 측정할 수 있는가?

03

자기 정보 (Self-Information)

Slide 4
정의
I(x) = −log₂ P(x)
P(x): 사건 x의 확률
직관
  • 낮은 확률 사건 → 더 많은 정보 제공
  • 높은 확률 사건 → 더 적은 정보 제공
  • 사건의 확률이 감소할수록 정보량 증가
확률 vs 정보량 예시
확률 P(x)정보량 I(x)
P(x) = 1.00 bits
P(x) = 0.51 bit
P(x) = 0.252 bits
P(x) = 0.1253 bits
Self-Information: I(x) = −log₂ P(x)
04

자기 정보의 성질

Slide 5
1
비음수성 (Non-negativity)
정보량은 항상 0 이상
0 < P(x) < 1 이면 log P(x) < 0
→ 음수 부호로 양수가 됨
I(x) ≥ 0
2
단조성 (Monotonicity)
확률이 낮을수록 더 많은 정보를 제공
희귀한 사건 = 더 많은 정보량
P(x₁) > P(x₂)
⟹ I(x₁) < I(x₂)
3
가산성 (Additivity)
독립 사건 x, y에 대해
전체 정보 = 개별 정보의 합
로그의 성질에서 유도됨
I(x,y) = I(x) + I(y)
📐
가산성 증명: −log P(x,y) = −log[P(x)·P(y)] = −log P(x) − log P(y) (독립일 때 P(x,y)=P(x)P(y))
05

엔트로피 (Entropy)

Slide 6–7
정의
H(X) = −Σ p(x) log p(x)
  • 자기 정보의 기댓값(Expected Value)
  • 확률 변수가 생성하는 평균 정보량
  • 전체 확률 분포의 불확실성 측정
버킷 예시 (Red/Blue balls)
버킷구성H(X)
#1 (100% Red)P(R)=1, P(B)=00 bits
#2 (75% Red)P(R)=0.75, P(B)=0.25≈ 0.811 bits
#3 (50/50)P(R)=0.5, P(B)=0.51 bit (최대)
🎯
이 개념은 Decision Tree에서 노드 불순도(impurity) 측정 및 최적 분할 결정에 핵심적으로 사용됨
베르누이 확률변수의 엔트로피 곡선
Entropy of a Bernoulli Random Variable
📊
엔트로피는 p=0.5일 때 최대(1 bit), p=0 또는 p=1일 때 최소(0 bit)
06

엔트로피의 성질

Slide 8
1
비음수성 (Non-negativity)
엔트로피는 항상 0 이상
확률은 0~1 사이이므로 엔트로피 값은 음수가 될 수 없음
H(X) ≥ 0
2
최대 엔트로피
모든 결과가 동일한 확률일 때 최대
n개의 결과가 동일 확률 1/n이면 최대 불확실성
H(X) = log n
(p(x) = 1/n 일 때)
3
결정적 경우 (0 엔트로피)
한 결과가 확률 1로 발생하면
불확실성이 없으므로 엔트로피 = 0
예: P(H)=1, P(T)=0
H(X) = 0
🪙
예시: 공정한 동전 P(H)=0.5, P(T)=0.5 → 최대 불확실성 H=1 bit / 편향된 동전 P(H)=1 → H=0 bit (완전 예측 가능)
07

조건부 엔트로피 (Conditional Entropy)

Slide 9
정의
H(Y|X) = −Σ p(x,y) log p(y|x)
H(Y|X) = Σ p(x) H(Y|X=x)
  • 다른 변수를 관측한 후 남은 불확실성 측정
  • H(Y|X): X를 알고 난 후 Y의 불확실성
  • X에 대한 기댓값으로 계산
해석
  • X가 Y를 잘 예측 → 조건부 엔트로피 작음
  • X가 Y에 대한 정보 거의 없음 → 조건부 엔트로피 큼
예시: 시험 결과 H(Y|X) 계산
Study (X)P(X)P(Pass|X)P(Fail|X)
Yes0.60.90.1
No0.40.20.8
계산 과정:
H(Y|X=Yes) = −(0.9 log 0.9 + 0.1 log 0.1) ≈ 0.469
H(Y|X=No) = −(0.2 log 0.2 + 0.8 log 0.8) ≈ 0.722
H(Y|X) = 0.6×0.469 + 0.4×0.722 ≈ 0.570
💡
공부 여부(X)를 알면 시험 결과(Y)의 불확실성이 감소
08

엔트로피 연쇄 법칙 (Chain Rule)

Slide 10
정의
H(X, Y) = H(X) + H(Y|X)
H(X, Y) = H(Y) + H(X|Y)
  • 결합 엔트로피 H(X,Y): 두 변수의 총 불확실성
  • 두 부분으로 분해 가능
  • ① 한 변수의 불확실성
  • ② 첫 번째 변수를 알고 난 후 나머지 변수의 불확실성
결합 엔트로피 분해
🔑
해석: 총 불확실성 = X의 불확실성 + X를 알고 난 후 Y의 나머지 불확실성
09

상호 정보 (Mutual Information)

Slide 11–12
정의
I(X;Y) = H(Y) − H(Y|X)
I(X;Y) = H(X) + H(Y) − H(X,Y)
  • 두 확률 변수가 공유하는 정보량
  • 한 변수를 알면 다른 변수의 불확실성이 얼마나 줄어드는지
  • 항상 비음수: I(X;Y) ≥ 0
극단 케이스
  • 독립인 경우: I(X;Y) = 0
    → 정보를 전혀 공유하지 않음
  • X가 Y를 완전히 결정하는 경우: I(X;Y) = H(Y)
    → Y의 모든 불확실성이 사라짐
  • 상호 정보가 클수록 변수 간 의존성이 강함
상호 정보 벤 다이어그램
📌
상호 정보는 대칭적: I(X;Y) = I(Y;X)
🔍
피처 선택 활용: max I(X;Y)를 최대화하는 피처 X를 선택 → 타겟 Y에 대한 가장 유익한 피처 선정
10

정보 이득 (Information Gain)

Slide 14
정의
IG = H(parent) − H(children)
IG = H(Y) − H(Y|X)
  • 피처를 기준으로 데이터셋을 분할한 후 엔트로피 감소량
  • H(Y): 타겟 변수의 엔트로피
  • H(Y|X): X를 관측한 후 남은 불확실성
  • 클래스 레이블 Y의 불확실성이
    피처 X를 알면 얼마나 줄어드는가
🌳
Decision Tree에서 각 노드에서 정보 이득이 가장 높은 피처를 선택하여 분기 기준으로 사용
Information Gain 시각화
11

결정 트리와 정보이론적 해석

Slide 15–16
핵심 원리
  • 목표: 클래스 레이블의 불확실성 감소
  • 각 노드에서 후보 피처 평가 후 최고 정보 이득을 가진 피처 선택
  • 정보이론적 관점: 상호 정보 최대화와 동치
  • 피처 X와 클래스 레이블 Y 사이의 max I(X;Y)
  • 결정 트리 학습 = 데이터셋의 불확실성을 반복적으로 감소시키는 과정
🎯
X* = argmax IG(Y|X)
최고 정보이득을 주는 피처 X*로 분기
Decision Tree 구조 예시 (age 기준 분기)
12

핵심 요약

📐
Self-Information
I(x) = −log₂ P(x)
단일 사건의 정보량. 확률이 낮을수록 정보량 증가. 비음수·단조·가산성 만족.
📊
Entropy
H(X) = −Σ p(x)log p(x)
확률 분포 전체의 평균 불확실성. 균등 분포일 때 최대, 결정적일 때 0.
🔀
Conditional Entropy
H(Y|X) = Σ p(x)H(Y|X=x)
X를 관측한 후 Y의 나머지 불확실성. Chain Rule: H(X,Y)=H(X)+H(Y|X).
🔗
Mutual Information
I(X;Y) = H(Y) − H(Y|X)
두 변수가 공유하는 정보량. I≥0, 독립이면 I=0, 완전결정이면 I=H(Y).
📈
Information Gain
IG = H(Y) − H(Y|X)
피처로 분할 후 엔트로피 감소량. Decision Tree의 최적 분기 기준으로 활용.
🌳
Decision Tree
X* = argmax I(X;Y)
정보이론 관점: 상호 정보 최대화 = 불확실성 반복 감소 과정.
핵심 개념 비교
개념 수식 측정 대상 값 범위 활용
Self-Information −log P(x) 단일 사건의 정보량 [0, ∞) 정보량 기초
Entropy H(X) −Σ p log p 분포 전체의 불확실성 [0, log n] 불순도 측정
Cond. Entropy H(Y|X) Σ p(x) H(Y|X=x) X 관측 후 Y의 불확실성 [0, H(Y)] 잔여 불확실성
Mutual Info I(X;Y) H(Y)−H(Y|X) 두 변수의 공유 정보 [0, min(H(X),H(Y))] 피처 선택
Information Gain H(Y)−H(Y|X) 분할 후 엔트로피 감소 [0, H(Y)] 결정 트리 분기
