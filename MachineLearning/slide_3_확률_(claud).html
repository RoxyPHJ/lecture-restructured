01
확률공간과 수학적 구조

확률이란 무엇인가?

확률(Probability)은 불확실한 사건(event)에 수치 [0,1]을 부여하는 함수입니다.

동전을 던지기 전 앞면이 나올 가능성을 어떻게 수치화할 수 있을까요?

빈도주의 (Frequentist)

P(A) = 실험을 무한히 반복할 때 A가 일어나는 상대 빈도

예) 동전 10,000번 → 앞면 5,023번
→ P(앞) ≈ 0.5023

⚠️ 한계: '태양이 내일 뜰 확률'처럼 반복 불가 사건에는 적용 어려움

베이지안 (Bayesian)

P(A) = 사건 A에 대한 주관적 믿음의 정도 (degree of belief)

예) '이 환자가 독감일 확률'처럼 사전 지식을 반영 가능

✅ 장점: 데이터 업데이트를 통해 믿음을 체계적으로 갱신 (Bayes)

확률 공간 (Probability Space) = (Ω, F, P)

Ω
표본 공간 (Sample Space)
가능한 모든 결과의 집합. '전체 세계'.
F
σ-대수 (σ-algebra)
측정 가능한 사건들의 집합. Ω의 부분집합 모음.
P
확률 측도 (Probability Measure)
F의 각 사건에 [0,1] 실수를 부여하는 함수.

예시 — 동전 2회 던지기

ΩF (사건의 예)P (확률 값)
{HH, HT, TH, TT} '앞면이 1번 이상' = {HH,HT,TH}
'앞면이 없음' = {TT}		 P({HH}) = 1/4
P(앞면 1번 이상) = 3/4

표본 공간 (Ω) 과 사건 (Event)

표본 공간 Ω

  • 확률 실험에서 나올 수 있는 모든 결과(outcome)의 집합
  • 원소 ω ∈ Ω 를 근원 사건(elementary event)이라 함
  • 동전: Ω = {H,T} / 주사위: Ω = {1,2,3,4,5,6} / 온도계: Ω = ℝ

사건 A ∈ F

  • 사건은 표본 공간의 부분집합 A ⊆ Ω
  • 여러 근원 사건의 묶음 — "앞면이 나왔다"
  • 전체 사건 Ω (반드시 일어남) / 공사건 ∅ (절대 불가)

사건의 집합 연산

A ∪ B
합사건 (Union)
A 또는 B가 일어나는 사건
A ∩ B
곱사건 (Intersection)
A와 B가 동시에 일어나는 사건
Aᶜ
여사건 (Complement)
A가 일어나지 않는 사건
P(Aᶜ) = 1−P(A)
A∩B=∅
배반사건 (Mutually Exclusive)
A와 B가 동시에 일어날 수 없음

σ-대수 (σ-algebra, σ-field) F

"어떤 사건에 확률을 부여할 수 있는가?"를 결정하는 수학적 구조입니다.

→ 간단히 설명하면 확률을 계산하는 함수 P가 입력 받을 수 있는 유효한 사건들의 집합

F = 2Ω : Ω가 유한 집합 / 자연수 / 정수 / 유리수 집합 일 때 → 가능한 모든 부분집합에 확률 부여 가능
F ⊆ 2Ω : Ω = ℝ 처럼 비가산 무한 집합 일 때 → 모든 부분집합에 일관된 확률 부여가 수학적으로 불가능함 → 확률을 줄 수 있는 사건만 골라 담은 것
단, 어떤 경우든 ∅ ∈ F, Ω ∈ F 는 항상 성립 → P(∅) = 0, P(Ω) = 1

확률 측도 P & Kolmogorov 공리 (1933)

P : F → [0,1]    사건(F의 원소)에 실수 값을 부여하는 함수
공리 1
비음수성 (Non-negativity)
P(A) ≥ 0   for all A ∈ F
확률은 음수가 될 수 없다.
공리 2
규범성 (Normalization)
P(Ω) = 1
전체 표본 공간의 확률은 반드시 1이다.
공리 3
가산 가법성 (σ-additivity)
A₁∩A₂=∅, A₁∩A₃=∅, … 이면  P(A₁∪A₂∪⋯) = P(A₁)+P(A₂)+⋯
서로 배반인 사건들의 합사건 확률 = 각 확률의 합

공리로부터 유도되는 중요 성질

  • ●
    P(∅) = 0
    불가능한 사건
  • ●
    P(Aᶜ) = 1 − P(A)
    여사건
  • ●
    P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
    포함-배제 원리
02
확률변수와 분포

확률변수 (Random Variable)

X : Ω → ℝ    사건 집합을 실수로 매핑하는 함수

핵심 개념: 확률변수 ≠ 변수, 확률변수 = 함수

  • 이름이 '변수'지만 실제로는 Ω → ℝ 방향의 함수
  • 확률 실험을 한 번 수행하면(ω를 뽑으면) 확률변수 X의 값 x = X(ω) 가 하나 결정됨
  • 확률변수 자체에는 확률이 없음 — P는 Ω 위에 정의되고, X를 통해 ℝ로 '밀려 내려오는' 것

이산 vs 연속 확률변수

이산 확률변수 (Discrete)

  • 취할 수 있는 값이 유한하거나, 자연수, 정수, 유리수 집합
  • X : Ω → {x₁, x₂, …}
  • 각 값에 양의 확률이 직접 부여됨

예시: 주사위 눈 {1,2,3,4,5,6} / 이메일 스팸 여부 {0,1} / 단어 출현 횟수 {0,1,2,...}

연속 확률변수 (Continuous)

  • 취할 수 있는 값이 실수 구간
  • X : Ω → ℝ (또는 부분 구간)
  • 특정 점의 확률은 0 → 구간으로만 정의

예시: 키, 몸무게, 온도, 전압 등 / 신경망 가중치 초기값

⚠️ 중요: (연속 확률변수에 대해서 특정 점의 확률) P(X = x) = 0
→ 연속 확률변수는 반드시 구간으로 확률을 계산: P(a ≤ X ≤ b) = ∫ₐᵇ f(x) dx

누적 분포 함수 CDF

F_X(x) = P(X ≤ x)

이산·연속 모두 정의 가능한 가장 기본적인 분포 표현

성질수식의미
단조 증가x₁<x₂ → F(x₁)≤F(x₂)x가 커질수록 누적
경계값F(-∞)=0, F(∞)=1전체 확률의 합은 1
우연속lim F(x+ε) = F(x)오른쪽에서 연속
구간 확률P(a<X≤b) = F(b)−F(a)두 CDF 값의 차이

이산의 경우: 계단 함수의 점프 크기 = 해당 점의 확률

확률 질량 함수 PMF (이산)

p(x) = P(X = x)

PMF의 필수 조건

  • p(x) ≥ 0 (모든 x에 대해)
  • Σₓ p(x) = 1 (전체 합이 1)
  • P(X ∈ A) = Σ_{x∈A} p(x)

예시 — 공정한 주사위

x123456
p(x)1/61/61/61/61/61/6

확률 밀도 함수 PDF (연속)

P(a ≤ X ≤ b) = ∫ₐᵇ f(x) dx    구간에 대한 확률을 면적으로 계산

PDF의 필수 조건 & CDF와의 관계

  • ①
    f(x) ≥ 0
    음수 불가
  • ②
    ∫₋∞^∞ f(x) dx = 1
    전체 면적 = 1
  • ③
    F_X(x) = ∫₋∞^x f(t) dt
    CDF = PDF의 적분
  • ④
    f(x) = dF(x)/dx
    PDF = CDF의 미분

PMF vs PDF 비교

PMF p(x)PDF f(x)
대상이산연속
값의 의미직접 확률확률 밀도
범위0 ≤ p(x) ≤ 1f(x) ≥ 0
(1 초과 가능)
전체Σ p(x) = 1∫ f(x)dx = 1
03
결합, 조건부, 독립

결합 확률 (Joint Probability)

p(x, y) = P(X = x, Y = y)    두 확률변수가 동시에 특정 값을 가질 확률

이산 결합 PMF

  • p(x, y) ≥ 0
  • Σₓ Σᵧ p(x, y) = 1
  • 두 확률변수를 동시에 모델링하는 확률 분포

연속 결합 PDF

  • f(x, y) ≥ 0
  • ∫∫ f(x, y) dx dy = 1
  • P(a≤X≤b, c≤Y≤d) = ∫∫ f(x,y) dxdy

예시 — 결합 PMF 표 (동전 2회, X=합, Y=최솟값)

p(x,y)Y=0Y=1Σᵧ (marginal)
X=01/401/4
X=11/201/2
X=201/41/4
Σₓ (marginal)3/41/41

주변 확률 (Marginal Probability)

결합 분포에서 나머지 변수를 '합산(이산)/적분(연속)해서 소거'하면 단일 변수의 분포를 얻을 수 있습니다.

이산 (Discrete)

p(x) = Σᵧ p(x, y)
p(y) = Σₓ p(x, y)

연속 (Continuous)

f(x) = ∫ f(x, y) dy
f(y) = ∫ f(x, y) dx

'소거'의 직관적 의미

X의 주변 분포 p(x): Y가 어떤 값이든 상관없이 X=x일 확률 → Y에 대해 '무관심하게' 통합

비유: 엑셀 표에서 행(X값)별로 열(Y값)을 모두 더해 행 합계를 구하는 것과 같습니다.

ML 연결

VAE(Variational Autoencoder)의 목표는 p(x) = ∫ p(x|z)p(z) dz 를 최대화하는 것 — z를 적분해 소거하는 주변화(marginalization)가 바로 이것입니다.

조건부 확률 (Conditional Probability)

P(A | B) = P(A ∩ B) / P(B)    단, P(B) > 0    B가 주어졌을 때 A의 확률

직관: 표본 공간을 B로 축소

'B가 일어났다'는 정보를 알게 되면 표본 공간이 Ω에서 B로 좁아집니다. 그 좁아진 공간에서 A가 차지하는 비율이 조건부 확률입니다.

연쇄 법칙 (Chain Rule of Probability)
P(A ∩ B) = P(A | B) · P(B) = P(B | A) · P(A)
일반화: P(x₁,x₂,…,xₙ) = P(x₁) · P(x₂|x₁) · P(x₃|x₁,x₂) · ··· · P(xₙ|x₁,…,xₙ₋₁)
개념수식설명
확률변수 버전p(x|y) = p(x,y) / p(y)y가 연속이면 p(y)>0 조건 필요
전확률 공식P(A) = Σₖ P(A|Bₖ)·P(Bₖ)Bₖ : Ω의 분할 → Bayes 분모의 계산 근거
ML 연결

언어 모델(GPT)의 핵심: P(w₁,w₂,...,wₙ) = ∏ₜ P(wₜ|w₁,...,wₜ₋₁) — 연쇄 법칙을 직접 모델링

독립성 (Independence) & 조건부 독립

P(A ∩ B) = P(A) · P(B)  ⟺  P(A|B) = P(A)  ⟺  P(B|A) = P(B)

독립의 의미

  • B가 일어났다는 정보가 A의 확률을 전혀 바꾸지 않는다 → 두 사건이 서로 '무관'
  • 독립 ≠ 배반 (Mutually Exclusive): 배반이면 오히려 종속 (P(A|B)=0 ≠ P(A))
조건부 독립 (Conditional Independence)
X ⊥ Y | Z  ⟺  P(X,Y|Z) = P(X|Z)·P(Y|Z)  ⟺  P(X|Y,Z) = P(X|Z)
Z를 알고 나면 X와 Y가 독립이 된다 — Z가 X,Y의 공통 원인일 때 자주 등장
⚠️ 독립과 조건부 독립은 다른 개념 — Z가 X와 Y의 결과일 경우 독립이지만 조건부 독립이 아닐 수 있음
독립 가정ML 활용
i.i.d. 가정
(independent & identically distributed)		X₁,…,Xₙ ∼ p(x) i.i.d. → MLE의 L(θ) = ∏ p(xᵢ|θ) 분해 근거
Naive Bayes 독립 가정P(x|y) = ∏ P(xᵢ|y) → 특징 간 조건부 독립 가정으로 계산 단순화

베이즈 정리 (Bayes' Theorem)

P(θ | X) = P(X | θ) · P(θ) / P(X) = Likelihood × Prior / Evidence
항 (Term)기호의미
사전 확률 (Prior)P(θ)데이터 관측 전 θ에 대한 믿음
우도 (Likelihood)P(X|θ)θ가 주어졌을 때 데이터의 확률
증거 (Evidence)P(X) = ∫P(X|θ)P(θ)dθ정규화 상수
사후 확률 (Posterior)P(θ|X)데이터 관측 후 갱신된 믿음

베이즈 정리 = 믿음의 업데이트 공식

사전 믿음 P(θ) + 데이터(X) 관측 → 사후 믿음 P(θ|X)로 업데이트

04
기댓값과 분산

기댓값 E[X] (Expectation / Mean)

이산 확률변수

E[X] = Σₓ x · p(x)

연속 확률변수

E[X] = ∫ x · f(x) dx

함수의 기댓값 E[g(X)]

E[g(X)] = Σₓ g(x)·p(x) (이산) 또는 ∫ g(x)·f(x) dx (연속)
  • E[X²]: X의 제곱 평균 → 분산 계산에 사용
  • E[aX + b] = aE[X] + b (기댓값의 선형성)

예) 공정한 주사위:

E[X] = 1·(1/6)+2·(1/6)+...+6·(1/6) = 21/6 = 3.5

주사위 눈 중 어느 것도 3.5가 아님 — 기댓값은 '가장 자주 나오는 값'이 아니라 '중심 무게'입니다.

기댓값의 중요 성질

선형성 (Linearity)

E[aX + bY + c] = aE[X] + bE[Y] + c

분포 형태와 무관하게 항상 성립 — 정의로부터 직접 유도

독립일 때 곱의 기댓값

X ⊥ Y → E[XY] = E[X] · E[Y]

독립이 아니면 성립 안 함에 주의

상수

E[c] = c

상수의 기댓값은 상수 자체

Jensen 부등식

f 가 볼록(convex)이면 f(E[X]) ≤ E[f(X)]

정보이론에서 KL 발산 ≥ 0 증명의 핵심

단조 기댓값: X ≥ 0 → E[X] ≥ 0 (비음수 확률변수의 기댓값은 비음수)

분산 & 표준편차

Var[X] = E[(X − μ)²] = E[X²] − (E[X])²  μ=E[X]

분산의 의미

  • 확률변수가 평균으로부터 얼마나 '퍼져 있는가'를 나타내는 척도
  • 단위: (원래 단위)² → 해석을 위해 표준편차 SD[X] = √Var[X] 사용

분산의 주요 성질

공식설명
Var[c] = 0상수는 퍼짐이 없음
Var[aX+b] = a²·Var[X]이동(+b)은 분산에 영향 없음, 스케일(a)은 제곱
X⊥Y 이면 Var[X+Y]=Var[X]+Var[Y]독립일 때만 분산의 합산 가능
s² = 1/(n−1) Σ(xᵢ−x̄)²표본 분산 — n-1로 나눔 → 불편 추정량 (Bessel 보정)

계산 공식: Var[X] = E[X²] − 2μE[X] + μ² = E[X²] − μ² = E[X²] − (E[X])²

핵심 유도 공식 모음

여사건

P(Aᶜ) = 1 − P(A)

포함-배제

P(A∪B) = P(A)+P(B)−P(A∩B)

조건부 확률

P(A|B) = P(A∩B) / P(B)

전확률 공식

P(A) = Σₖ P(A|Bₖ)·P(Bₖ)

베이즈 정리

P(θ|X) ∝ P(X|θ)·P(θ)

분산 계산 공식

Var[X] = E[X²] − (E[X])²
