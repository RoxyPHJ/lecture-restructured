기초통계 개요
통계학 (Statistics)
  • 자료를 정보화하여 객관적 의사결정을 연구하는 학문
  • 기계학습에서는 데이터의 통계적 속성이 모델 설계와 학습의 핵심 근거
용어 정의 기계학습 연결
모집단 (Population)관심 대상 전체 데이터학습 데이터가 추출된 실제 분포
표본 (Sample)모집단의 일부 수집 데이터학습 데이터셋 (Training Set)
요약 (Summary)데이터를 정보 손실 최소화하여 정리특징 벡터 추출, 통계량 계산

02
통계 파라미터
평균 (Mean)
  • 자료의 총합을 자료의 개수로 나눈 것
  • 자료의 분포를 질점계로 보면 무게 중심에 해당
x̄ = (1/n) · Σᵢ xᵢ
분산 (Variance)
  • 평균값과의 차이에 대한 제곱값의 평균
  • 자료의 흩어진 정도를 나타냄
s² = (1/(n-1)) · Σᵢ (xᵢ - x̄)²
표준편차 (Standard Deviation)
  • 분산의 제곱근을 취하여 자료의 단위와 일치시킨 것
s = √[ (1/(n-1)) · Σᵢ (xᵢ - x̄)² ]

핵심 세 통계량의 관계

  • 평균: 데이터의 중심 위치
  • 분산: 중심에서 퍼진 정도 (단위² )
  • 표준편차: 분산의 제곱근 (원래 단위)
  • 분모 n-1: 불편추정량 (Bessel 보정)
03
바이어스 & Bias–Variance Tradeoff
바이어스 (Bias)

데이터의 편향된 정도를 나타냄

Bias–Variance Tradeoff
기대 오차 분해:
E[(y - f̂(x))²] = (E[f̂(x)] - f(x))² + E[(f̂(x) - E[f̂(x)])²] + δ
= Bias² + Variance + Noise
Bias²

모델 평균 예측값이 실제 정답에서 얼마나 벗어나는가. 모델이 너무 단순할 때 높아짐 → Underfitting

Variance

학습 데이터가 바뀔 때 예측값이 얼마나 흔들리는가. 노이즈까지 외울 때 높아짐 → Overfitting

Noise

데이터 자체에 내재된 불확실성. 어떤 모델로도 줄일 수 없는 하한값

핵심: Model Complexity ↑ → Bias ↓, Variance ↑ → 두 오차 사이에는 Trade-off 관계가 있다.
04
공분산 & 상관계수
공분산 (Covariance)

두 개 이상의 확률변수가 주어질 경우 각 변수 간의 변화하는 양상을 나타내는 통계적 척도

x̄ = (1/n)Σxᵢ, ȳ = (1/n)Σyᵢ

C(x,y) = 1/(n-1) · Σᵢ (xᵢ - x̄)(yᵢ - ȳ)

= 1/(n-1) · (Σxᵢyᵢ - Σxᵢȳ - Σx̄yᵢ + nx̄ȳ)
상관계수 (Correlation Coefficient)
  • 두 확률변수 X, Y 사이의 상관관계 정도를 나타내는 수치: 0 ≤ |ρ| ≤ 1
  • σₓ, σᵧ는 변량 x, y의 표준편차값
ρ = C(x,y) / (σₓ · σᵧ) = C(x,y) / (√V(x) · √V(y))
작은 상관계수는 선형적으로 두 성분 간의 관련성이 거의 없음을 나타낸다.

왜도 & 첨도
왜도 (Skewness)
  • 확률 분포가 어느 한쪽으로 치우친 비대칭(asymmetry) 정도를 나타내는 척도
  • 오른쪽으로 더 길면 양의 값, 왼쪽으로 더 길면 음의 값, 좌우 대칭이면 0
Skewness = Σᵢ(xᵢ - x̄)³ / [Σᵢ(xᵢ - x̄)²]^(3/2)
첨도 (Kurtosis)
  • 분포의 뾰족한(peakedness) 정도를 나타내는 통계적 척도
Kurtosis = Σᵢ(xᵢ - x̄)⁴ / [Σᵢ(xᵢ - x̄)²]²
< 3
평편한 분포
(Platykurtic)
= 3
정규분포
(Mesokurtic)
> 3
뾰족한 분포
(Leptokurtic)
06
회귀 분석 (최소제곱법)
회귀직선 & 최소제곱법
  • 표본을 대표하는 직선 ŷ = θ₀ + θ₁x 를 찾는 것
// 비용 함수 (MSE)
J(θ) = (1/N) · Σᵢ (ŷᵢ - yᵢ)²
// 정규방정식 (Normal Equation)
θ = (XᵀX)⁻¹ Xᵀy
XᵀX가 특이행렬(singular)이면 역행렬 불가
→ Ridge 정규화: (XᵀX + λI)⁻¹Xᵀy
기계학습 관점에서의 회귀
  • 최소제곱법 = MSE 손실의 해석적 해(closed-form)
  • 딥러닝에서는 경사하강법(Gradient Descent)으로 근사
  • 다항 회귀: 비선형 관계를 고차원 특징으로 선형 표현 → 과적합(overfitting) 위험
// 신경망의 출력층 = 회귀의 일반화
ŷ = f(x; θ)
// f: 비선형 함수 구성
07
정규분포 & 중심극한정리
정규분포 (Normal/Gaussian)
  • 기계학습에서 가장 광범위하게 사용되는 연속 확률 분포
  • μ: 평균 (분포의 위치)
  • σ²: 분산 (분포의 너비)
N(x;μ,σ²) = (1/√(2πσ²)) · exp(-(x-μ)² / 2σ²)
68-95-99.7 규칙
P(μ±σ) ≈ 68%
P(μ±2σ) ≈ 95%
P(μ±3σ) ≈ 99.7%
중심극한정리 (Central Limit Theorem)
  • X₁, X₂, ... Xₙ ~ p(x), E[Xᵢ]=μ, Var[Xᵢ]=σ² 이고, Xᵢ들이 독립 단일 분포에서 샘플링되었을 때
  • 표본평균 X̄ₙ = (1/n)·Σᵢ Xᵢ 는 n→∞ 이면 N(μ, σ²/n)을 따름
  • 원래 분포의 형태에 무관하게 성립
  • 보통 n ≥ 30 부터 정규분포를 따른다고 볼 수 있음
X̄ₙ = (1/n) · Σᵢ Xᵢ → N(μ, σ²/n) (n→∞)
ML 활용: 오차 모델링의 근거 / 미니배치 학습에서 배치 평균의 분포 보장
08
MLE — 최대우도추정 수식 유도
MLE의 Parameter θ
  • θ: 원본 확률 분포를 계산(근사)하기 위해 추정하는 파라미터
  • 여러 확률 밀도 함수 중 관측 x₁, x₂, ..., xₙ 과 제일 부합하는 함수를 찾는 과정
  • 원본 분포가 Gaussian일 때: θ = (μ, σ²)
  • 딥러닝 네트워크: θ = 인공지능 네트워크의 가중치
① 우도 함수 (Likelihood Function)
L(θ; X) = p(x₁, x₂, ..., xₙ | θ) = ∏ᵢ p(xᵢ | θ)
← i.i.d. 가정으로 곱 분리
i.i.d. (independent identically distributed): 샘플한 확률 변수들이 서로 같은 확률 분포에서 독립적으로 샘플링되었다는 가정
② 로그 우도 (Log-Likelihood)
ℓ(θ) = log L(θ; X) = Σᵢ log p(xᵢ | θ)
← log는 단조 증가 → argmax 보존, 곱 → 합, 수치 안정성 ↑
③ MLE 추정량
θ̂ₘₗₑ = argmax_θ ℓ(θ)

// 필요조건 (1계 미분 = 0)
∇_θ ℓ(θ) = 0
④ Gaussian에 MLE 적용 — 표본 평균·분산이 MLE 추정량
ℓ(μ,σ²) = -(n/2)·log(2πσ²) - Σ(xᵢ-μ)² / (2σ²)

∂ℓ/∂μ = 0 → μ̄ = (1/n)·Σxᵢ (표본 평균)
∂ℓ/∂σ² = 0 → σ̄² = (1/n)·Σ(xᵢ - μ̄)² (표본 분산)
정규분포 하의 MSE 최소화 = MLE와 동치 (log-likelihood 최대화)

MAP 추정 & 정규화
MAP (Maximum A Posteriori)
θ_MAP = argmax P(θ|X) = argmax [log P(X|θ) + log P(θ)]
= likelihood + prior
MLE vs MAP 비교
MLEMAP
목적함수argmax Σ log p(xᵢ|θ)argmax [Σ log p(xᵢ|θ) + log P(θ)]
Prior 사용✗ 사용 안 함✓ Prior P(θ) 반영
데이터 적을 때불안정 (과적합 위험)Prior가 안전장치 역할
Prior → 0—MAP = MLE (Prior 무관)
MAP = 정규화 (Regularization)의 수학적 근거
Prior P(θ)log P(θ)MAP 목적함수동치인 정규화
Gaussian N(0, 1/λ) -λ/2 · ||θ||² Σ log p(xᵢ|θ) - λ||θ||² L2 정규화 (Ridge)
Laplace L(0, 1/λ) -λ · ||θ||₁ Σ log p(xᵢ|θ) - λ||θ||₁ L1 정규화 (Lasso)

확률 분포 (이산 / 연속)
주요 이산 확률 분포
분포PMFE[X] / Var[X]ML 활용
베르누이 Ber(p) P(X=1)=p, P(X=0)=1-p E=p, Var=p(1-p) 이진 분류 출력 / Sigmoid 함수
이항 Bin(n,p) C(n,k)·pᵏ·(1-p)ⁿ⁻ᵏ E=np, Var=np(1-p) 베르누이 분포의 n회 독립 시행 성공 횟수
카테고리형 Cat(π) P(X=k)=πₖ, Σπₖ=1 E[1ₓ₌ₖ]=πₖ 다중 클래스 분류 / Softmax 출력
포아송 Poi(λ) P(X=k)=λᵏe⁻λ/k! E=λ, Var=λ 단위 시간당 이벤트 발생 횟수
정규분포 (Gaussian) N(μ, σ²) — ML에서 가장 중요
f(x; μ,σ²) = (1/√(2πσ²)) · exp(-(x-μ)² / (2σ²))
최대 엔트로피 분포: 평균·분산이 고정될 때 가장 불확실한 분포 / CLT에 의해 자연·공학 전반에 등장
기타 주요 연속 분포
분포PDF 핵심ML 활용
균등 Uniform U(a,b) f(x) = 1/(b-a) · [a,b] 구간 무정보 사전 분포 / 초기 난수 생성
라플라스 L(μ,b) f(x) = exp(-|x-μ|/b) / (2b) L1 정규화 = Laplace Prior MAP / 희소 가중치 유도
다변량 Gaussian N(μ,Σ) exp(-½(x-μ)ᵀΣ⁻¹(x-μ)) / |2πΣ|^½ GAN·VAE·GMM 클러스터링 / 공분산 행렬 Σ 포함

비교 주요 분포 형태 비교

  • ■ Gaussian (μ=0, σ=1)
  • ■ Laplace (μ=0, b=1)
  • ■ Uniform (-2, 2)
Laplace는 Gaussian보다 꼬리가 두껍고 중심이 뾰족해 희소성(sparsity)을 유도

생성 vs 판별 모델 & 베이즈 정리
판별 모델 (Discriminative)
학습 목표: P(y|x) 조건부 분포 직접 학습
  • 결정 경계만 학습 → 분류 정확도 높음
  • 새로운 데이터 생성 불가
예시:
Logistic Regression SVM 신경망 분류기 Transformer
생성 모델 (Generative)
학습 목표: P(x,y) = P(x|y)·P(y) 결합 분포 학습
  • 데이터 생성 분포 자체를 모델링
  • 새로운 데이터 생성 가능
예시:
Naive Bayes GMM VAE GAN Diffusion LLM
연쇄 법칙 (Chain Rule) — 언어 모델의 핵심

P(x₁,x₂,...,xₙ) = P(x₁)·P(x₂|x₁)·P(x₃|x₁,x₂)···P(xₙ|x₁,...,xₙ₋₁)

LLM의 자기회귀 생성:
P(w₁,...,wₜ) = ∏ₜ P(wₜ|w₁,...,wₜ₋₁) → 이전 토큰들을 조건으로 다음 토큰 예측
베이즈 정리와 기계학습 — MLE·MAP·베이지안 학습
P(θ|X) = P(X|θ)·P(θ) / P(X) = Likelihood × Prior / Evidence
MLE (최대 우도)
θ̂ = argmax P(X|θ)
  • Prior 무시
  • 데이터가 많을수록 효과적
  • L(θ) = ∏p(xᵢ|θ) 최대화
MAP (최대 사후 확률)
θ̂ = argmax P(X|θ)·P(θ)
  • Prior 반영 → 정규화 효과
  • Gaussian Prior → L2
  • Laplace Prior → L1
베이지안 추론
전체 분포 P(θ|X) 활용
  • 분포 전체 유지
  • 불확실성 정량화 가능
  • 계산 비용 높음 (MCMC)
실제 ML에서의 선택: 대부분의 딥러닝 = MLE (데이터 충분) / 정규화 필요 = MAP / 불확실성 중요 = 베이지안 신경망

핵심 요약
01
평균·분산·표준편차
데이터의 중심과 퍼짐 정도. 분모 n-1로 불편추정량 계산.
02
Bias-Variance Tradeoff
총 오차 = Bias² + Variance + Noise. 복잡도 ↑ → Bias↓ Variance↑
03
공분산·상관계수
변수 간 관계. ρ ∈ [-1,1]. 부호는 방향, 크기는 선형 관계 강도.
04
왜도·첨도
왜도: 비대칭 정도 (0=대칭). 첨도: 뾰족한 정도 (정규분포=3)
05
회귀 / 최소제곱법
θ=(XᵀX)⁻¹Xᵀy. MSE 최소화 = MLE. 딥러닝에서는 경사하강법 사용.
06
정규분포 & CLT
N(μ,σ²). 중심극한정리: n≥30이면 표본평균이 정규분포 따름.
07
MLE
θ_MLE = argmax Σ log p(xᵢ|θ). i.i.d. 가정. Gaussian → 표본평균·분산.
08
MAP & 정규화
Prior 추가. Gaussian Prior → L2(Ridge). Laplace Prior → L1(Lasso).
09
확률 분포
Bernoulli→sigmoid, Categorical→softmax, Gaussian→연속 출력, Laplace→sparsity
10
생성 vs 판별 모델
판별: P(y|x) 학습 (분류 정확). 생성: P(x,y) 학습 (데이터 생성). LLM: 연쇄법칙.
