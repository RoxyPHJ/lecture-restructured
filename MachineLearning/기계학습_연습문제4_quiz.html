01
주성분 분석 (PCA)
선형 종속 구조 · 공분산 행렬 · 고유값 분해 · 차원 축소
원본 문제 요약
3×3 데이터 행렬, x₃ = x₁ + x₂
X =
[1 1 2] [2 0 2] [3 1 4] x₃ = x₁ + x₂ (완전 선형 종속)
→ 평균 제거 → 공분산 → 표준화 → 상관계수 행렬 → 고유값 분해 → 2D 투영
[응용] PCA — 1-A
⭐⭐ 보통 수치 변형 계산형
P1-A
아래 데이터 행렬 X′로 동일 절차를 수행하시오.
X′ = [2 1 3] [4 2 6] [0 3 3] 단, x₃ = x₁ + x₂ 관계가 여전히 성립함을 먼저 확인하시오.
(가)
각 특성의 표본 평균 μ = (μ₁, μ₂, μ₃)와 평균 제거 행렬 X̃를 구하시오.
(나)
공분산 행렬 Σ = (1/N)X̃ᵀX̃ 를 계산하고, Cov(x₁, x₂)의 값과 원본 문제와의 차이를 서술하시오.
(다)
정규화 행렬 Z와 상관계수 행렬 C를 구하시오.
(라)
det(C − λI) = 0을 풀어 고유값을 구하시오. 0이 아닌 고유값이 몇 개인지, 그 이유를 x₃와의 관계로 설명하시오.
(마)
2차원으로 축소할 때 정보 보존율을 계산하고, 투영 결과 Y = ZW를 구하시오.
풀이
(가) 평균 및 평균 제거
μ₁ = (2+4+0)/3 = 2, μ₂ = (1+2+3)/3 = 2, μ₃ = (3+6+3)/3 = 4
X̃ = [ 0 -1 -1 ] [ 2 0 2 ] [-2 1 -1 ]
(나) 공분산 행렬
Σ = (1/3)X̃ᵀX̃ Σ₁₁ = (0²+2²+(-2)²)/3 = 8/3 Σ₂₂ = ((-1)²+0²+1²)/3 = 2/3 Σ₃₃ = ((-1)²+2²+(-1)²)/3 = 2 = Σ₁₁+Σ₂₂ (선형종속 확인) Cov(x₁,x₂) = (0·(-1)+2·0+(-2)·1)/3 = -2/3 ≠ 0
→ 원본에서는 Cov(x₁,x₂) = 0 이었으나, 여기서는 −2/3으로 x₁, x₂ 사이에 음의 상관이 존재함. (다) 표준화 행렬 Z, 상관계수 행렬 C
σ₁ = √(8/3), σ₂ = √(2/3), σ₃ = √2 Z_ij = X̃_ij / σ_j (각 열을 표준편차로 나눔) C = (1/3)ZᵀZ (3×3 상관계수 행렬)
(라) 고유값
x₃ = x₁ + x₂ → 완전 선형 종속 → rank(C) ≤ 2 → λ₃ = 0 반드시 존재.
비영 고유값 개수: 2개
(구체적 값은 C의 대각 원소와 Cov를 이용해 특성방정식 전개로 도출) (마) 정보 보존율 및 투영
보존율 = (λ₁ + λ₂) / (λ₁ + λ₂ + 0) = 100% (λ₃ = 0 이므로 2D 투영 시 정보 손실 없음)
W = [v₁ | v₂]로 Y = ZW 계산 (3×2 행렬)
변형 포인트: 원본 Cov(x₁,x₂) = 0이었으나, 이 문제는 Cov(x₁,x₂) ≠ 0으로 변경. x₁, x₂가 독립이 아니어도 x₃ 선형 종속이 유지되면 고유값 0은 동일하게 발생함을 학습.
원본 문제 요약
2D 축소: 분산 설명률 계산 후 2D 투영
3개의 고유값 중 상위 2개 선택 → 정보 보존율 제시 → Y = ZW (3×2)
[응용] PCA — 1-B
⭐⭐ 보통 역방향 문제 계산형
P1-B
고유값과 분산 설명률이 주어졌을 때, 원본 데이터의 특성을 역추론하시오.
어떤 3차원 데이터에 대해 상관계수 행렬 C의 고유값이 다음과 같이 계산되었다:
λ₁ = 2.5, λ₂ = 0.5, λ₃ = 0
(가)
각 고유값의 분산 설명률과, 2D로 축소할 때의 정보 보존율을 구하시오.
(나)
λ₃ = 0이 의미하는 바를 데이터의 특성 구조 측면에서 설명하시오. 이 데이터에 반드시 존재하는 특성 간 관계식의 형태를 서술하시오.
(다)
만약 λ₃ = 0.1 (≈ 0)으로 바뀐다면, 2D 축소 시 정보 보존율과 손실 정보량은 각각 몇 %인가? λ₃가 완전히 0이 아닐 때 발생하는 실용적 문제를 설명하시오.
(라)
1D로만 축소할 때 분산 설명률은 얼마인가? 원본 문제(2D 축소)와 비교하여 차원 축소 깊이 선택 기준을 논하시오.
풀이
(가) 분산 설명률
총합 = 2.5 + 0.5 + 0 = 3.0 λ₁ 설명률 = 2.5/3.0 = 83.3% λ₂ 설명률 = 0.5/3.0 = 16.7% λ₃ 설명률 = 0/3.0 = 0% 2D 보존율 = (2.5+0.5)/3.0 = 100%
(나) λ₃ = 0의 의미
상관계수 행렬의 rank < 3 → 특성 간 완전 선형 종속 존재.
어떤 특성 xₖ가 다른 특성들의 선형결합으로 표현됨:
xₖ = a·xᵢ + b·xⱼ (a, b는 상수)
데이터가 실제로는 2차원 초평면(hyperplane) 위에 놓여 있음. (다) λ₃ = 0.1일 때
총합 = 2.5 + 0.5 + 0.1 = 3.1 2D 보존율 = (2.5+0.5)/3.1 ≈ 96.8% 손실 = 0.1/3.1 ≈ 3.2%
실용적 문제: 어떤 특성 조합이 선형 종속인지 불명확 → 주성분 수 결정 임계값(예: 95%) 기준 설정 필요. (라) 1D 축소 시
1D 보존율 = 2.5/3.0 = 83.3% 2D 대비 추가 설명량 = 0.5/3.0 = 16.7%p 손실
기준: 보존율 ≥ 95%를 목표로 할 때 2D가 필요; 시각화가 목적이라면 2D가 표준.
변형 포인트: 고유값 → 투영 순서를 역전. 원본은 데이터→고유값 방향이나, 여기서는 고유값→데이터 구조 추론의 역방향 문제.
원본 문제 요약
x₃ = x₁ + x₂ (계수 1:1) 선형 종속
x₃는 x₁과 x₂의 단순 합. PCA 후 2D 보존율 = 100%.
[응용] PCA — 1-C
⭐⭐⭐ 어려움 개념 결합 계산형 + 종합판단형
P1-C
x₃ = 2x₁ − x₂인 데이터에 대해 PCA를 수행하고, 원본 문제와의 차이를 분석하시오.
X″ = [1 2 0] [3 0 6] [2 4 0] 단, x₃ = 2x₁ − x₂ 관계를 먼저 검증하시오.
(가)
μ와 X̃를 계산하시오.
(나)
공분산 행렬 Σ를 계산하시오. Cov(x₁, x₂)의 부호와 그 직관적 의미를 설명하시오.
(다)
다음 두 주장 중 옳은 것을 고르고, 이유를 고유값 관점에서 서술하시오.
  ① 이 데이터도 2D 투영 시 100% 정보 보존된다.
  ② x₃ = 2x₁ − x₂이므로 원본 문제보다 보존율이 낮다.
(라)
만약 이 데이터에 노이즈(noise)를 추가하여 x₃ ≈ 2x₁ − x₂ + ε (ε: 소량 오차)로 만들면, λ₃는 어떻게 변하는가? 이것이 실제 머신러닝에서 PCA를 사용하는 이유와 어떻게 연결되는지 설명하시오.
풀이
(가) 평균 및 평균 제거
μ₁ = (1+3+2)/3 = 2, μ₂ = (2+0+4)/3 = 2, μ₃ = (0+6+0)/3 = 2 검증: 2·1−2=0 ✓, 2·3−0=6 ✓, 2·2−4=0 ✓ X̃ = [-1 0 -2] [ 1 -2 4] [ 0 2 -2]
(나) 공분산 행렬
Σ₁₁ = (1+1+0)/3 = 2/3 Σ₂₂ = (0+4+4)/3 = 8/3 Cov(x₁,x₂) = (-1·0 + 1·(-2) + 0·2)/3 = -2/3 < 0
→ x₁이 커질 때 x₂는 작아지는 경향 (x₃=2x₁−x₂에서 x₁↑ → 2x₁↑ → x₂ 상대적으로 작아야 x₃ 일정 유지) (다) 올바른 주장: ①
x₃ = 2x₁ − x₂도 완전 선형 종속 → rank(C) ≤ 2 → λ₃ = 0
계수(1:1 vs 2:−1)는 선형 종속 여부에 영향 없음. 2D 보존율 = 100%. (라) 노이즈 추가 시
ε ≠ 0 → x₃가 x₁, x₂의 완전 선형결합에서 벗어남 → λ₃ > 0 (매우 작은 양수로 변화) → 2D 보존율 < 100% (예: 99.x%)
실제 데이터는 항상 노이즈 포함 → λ₃ > 0가 일반적. PCA는 작은 λ를 가진 방향(노이즈 방향)을 제거하여 노이즈 제거 + 차원 축소를 동시에 달성. 이것이 실제 ML에서 PCA를 전처리로 사용하는 핵심 이유.
변형 포인트: 계수를 x₃=x₁+x₂ → x₃=2x₁−x₂로 변경. 계수 변형이 고유값 0 여부에 영향이 없음을 확인하고, 노이즈 개념까지 연결하는 개념 결합 문제.
02
Decision Tree — Gini 불순도
분기 기준 선택 · 가중 Gini · Gini 감소량 · 트리 구조
원본 문제 요약
10명 환자 (종양크기/흡연기간 → 양성/악성)
양성 4명 / 악성 6명
종양크기(소 5명, 대 5명) vs 흡연기간(단기 4명, 장기 6명)
루트: 종양크기 선택 → max depth 2
[응용] DT — 2-A
⭐⭐ 보통 수치 변형 계산형
P2-A
아래 12명의 환자 데이터로 동일 절차를 수행하시오.
환자종양크기흡연기간진단
1소단기양성
2소단기양성
3소단기양성
4소단기양성
5소장기악성
6소장기악성
7대단기악성
8대장기악성
9대장기악성
10대장기악성
11대장기악성
12대장기악성
조건: Gini 불순도 기준, max depth = 2
(가)
전체 데이터셋의 Gini(S)를 계산하시오.
(나)
종양크기와 흡연기간 각각으로 분기할 때의 가중 Gini 불순도와 Gini 감소량을 계산하고, 루트 노드 분기 기준을 결정하시오.
(다)
순수하지 않은 자식 노드에서 깊이 2 분기를 수행하고 Gini 감소량을 계산하시오.
(라)
최종 트리 구조를 텍스트로 나타내고, 새 환자 {종양크기=소, 흡연기간=장기} 의 예측 결과를 제시하시오.
풀이
(가) 전체 Gini
양성 4명, 악성 8명 (총 12명) Gini(S) = 1 − ((4/12)² + (8/12)²) = 1 − (1/9 + 4/9) = 1 − 5/9 = 4/9 ≈ 0.444
(나) 분기 기준 비교
종양크기 기준:
소 (6명): 양성 4, 악성 2 → Gini = 1−((4/6)²+(2/6)²) = 1−(4/9+1/9) = 4/9 대 (6명): 양성 0, 악성 6 → Gini = 0 (순수!) 가중 Gini = (6/12)·(4/9) + (6/12)·0 = 2/9 ≈ 0.222 Gini 감소량 = 4/9 − 2/9 = 2/9 ≈ 0.222
흡연기간 기준:
단기 (5명): 양성 4, 악성 1 → Gini = 1−((4/5)²+(1/5)²) = 1−(16/25+1/25) = 8/25 장기 (7명): 양성 0, 악성 7 → Gini = 0 (순수!) 가중 Gini = (5/12)·(8/25) + (7/12)·0 = 40/300 = 2/15 ≈ 0.133 Gini 감소량 = 4/9 − 2/15 = 20/45 − 6/45 = 14/45 ≈ 0.311
→ 흡연기간 선택 (Gini 감소량 더 큼) (다) 깊이 2 분기
단기 노드 (양성 4, 악성 1) → 종양크기로 추가 분기:
소/단기: 양성 4, 악성 0 → Gini = 0 ✓ 대/단기: 양성 0, 악성 1 → Gini = 0 ✓ Gini 감소량 = 8/25 − 0 = 0.320
(라) 최종 트리 구조
[루트] 흡연기간? ├─ 단기 (5명) │ └─ 종양크기? │ ├─ 소 → [양성] (4명) │ └─ 대 → [악성] (1명) └─ 장기 (7명) → [악성] (순수 노드)
새 환자 {소, 장기}: 흡연기간=장기 → 악성
변형 포인트: 샘플 수 10→12명, 종양크기 소 그룹에 양성 1명 추가(소·단기:4명). 루트 분기 기준이 원본(종양크기)에서 흡연기간으로 변경되어 분기 순서가 역전됨.
원본 문제 요약
max depth = 2, 두 특성 (종양크기, 흡연기간)
루트: 종양크기 → 깊이 2: 흡연기간
새 환자 예측: {소, 장기} → 악성
[응용] DT — 2-B
⭐⭐ 보통 조건 변형 결과 도출형
P2-B
원본 데이터에 세 번째 특성 "나이(40대 이하 / 50대 이상)"를 추가하시오.
원본 10명 데이터에 아래 나이 정보를 추가한다:
환자종양크기흡연기간나이진단
1~3소단기40대↓양성
4~5소장기50대↑악성
6대단기50대↑악성
7~10대장기50대↑악성
조건: max depth = 1 (루트 노드만 분기, 즉 Stump)
(가)
종양크기, 흡연기간, 나이 세 특성 각각의 Gini 감소량을 계산하시오.
(나)
가장 Gini 감소량이 큰 특성을 루트 분기로 선택하시오. 원본 문제(종양크기 선택)와 결과가 같은가, 다른가? 다르다면 왜인지 설명하시오.
(다)
max depth = 1로 제한되어 있으므로, 이 Decision Stump의 오분류율(error rate)을 계산하시오.
(라)
max depth를 1→2로 늘렸을 때 오분류율이 감소하는가? 이것이 Random Forest의 편향-분산 트레이드오프와 어떻게 연결되는지 설명하시오.
풀이
(가) 세 특성 Gini 감소량
전체: 양성 3, 악성 7 → Gini(S) = 1 − (9/100 + 49/100) = 42/100 = 0.42
종양크기: 소(5): 양성3, 악성2 → Gini=1−(9/25+4/25)=12/25=0.48 대(5): 양성0, 악성5 → Gini=0 가중Gini=(5/10)·0.48+(5/10)·0 = 0.24 감소량=0.42−0.24=0.18 흡연기간: 단기(4): 양성3, 악성1 → Gini=1−(9/16+1/16)=6/16=0.375 장기(6): 양성0, 악성6 → Gini=0 가중Gini=(4/10)·0.375=0.15 감소량=0.42−0.15=0.27 나이: 40대↓(3): 양성3, 악성0 → Gini=0 50대↑(7): 양성0, 악성7 → Gini=0 가중Gini=0 감소량=0.42−0=0.42
(나) 나이 선택 (감소량 0.42 최대)
원본(종양크기, 0.18)과 다름. 나이가 양성/악성을 완벽 분리 → 이 데이터셋에서 가장 강력한 특성. (다) 오분류율
나이로 분기 시: 40대↓ → 양성 예측: 3명 모두 양성 ✓ 50대↑ → 악성 예측: 7명 모두 악성 ✓ 오분류율 = 0/10 = 0%
(나이가 완벽 분리자이므로 depth=1만으로도 오분류 없음) (라) depth 증가와 편향-분산 트레이드오프
depth 증가 → 훈련 오분류율 감소(편향↓) but 과적합 위험(분산↑).
Random Forest: 여러 depth가 큰 트리를 각각 다른 부트스트랩 샘플로 학습 → 개별 트리의 고분산을 앙상블로 상쇄 → 낮은 편향 + 낮은 분산 동시 달성.
변형 포인트: 특성 2개 → 3개로 추가, max depth=2→1로 제한. 완벽 분리자(나이)가 도입되어 루트 선택이 완전히 바뀌는 상황을 학습.
원본 문제 요약
Gini 감소량으로 루트 선택 → 깊이 2까지 분기
루트(종양크기) 분기 후 "소" 그룹이 불순 → 흡연기간으로 추가 분기
[응용] DT — 2-C
⭐⭐⭐ 어려움 역방향 문제 종합판단형
P2-C
아래 Decision Tree 구조가 주어졌을 때, 이 트리를 만든 원본 데이터를 역추론하시오.
[루트] 흡연기간? ├─ 단기 │ └─ 종양크기? │ ├─ 소 → [양성] (n=a) │ └─ 대 → [악성] (n=b) └─ 장기 → [악성] (n=c, 순수 노드)
조건:
  • 전체 샘플 N = 12명
  • 양성 : 악성 = 1 : 2
  • 루트 Gini(S) = 4/9

(가)
a, b, c의 가능한 값을 조건을 이용해 결정하시오. (힌트: 양성 4명, 악성 8명)
(나)
루트에서 흡연기간으로 분기하는 것이 종양크기로 분기하는 것보다 Gini 감소량이 크려면, 단기 그룹의 악성 비율이 어떤 조건을 만족해야 하는가? 부등식으로 표현하시오.
(다)
이 트리로 새 환자를 분류할 때의 예측 결과를 구하시오:
① {단기, 소}    ② {단기, 대}    ③ {장기, 소}
(라)

풀이

N=12, 양성:악성=1:2 → 양성 4명, 악성 8명 장기 그룹(c명): 악성만 → c명 모두 악성 단기 그룹(12−c명): 양성 a + 악성 b = 12−c 장기 그룹이 순수 악성이고 단기에 양성이 모두 있으므로: a = 4 (전체 양성이 단기에만 있음) 가장 단순한 경우: c = 6, 단기 = 6명 → a = 4, b = 2, c = 6
(나) 흡연기간이 종양크기보다 Gini 감소량이 크려면
설 단기 n₁명, 그 중 악성 비율 = p Gini(단기) = 1 − ((1−p)² + p²) = 2p(1−p) 가중 Gini(흡연기간) = (n₁/12)·2p(1−p) + 0 종양크기 기준 가중 Gini와 비교하여 흡연기간 감소량 > 종양크기 감소량 ⟺ 단기 그룹의 불순도가 충분히 낮아야 함 ⟺ p < p* (임계값) 구체적으로 종양크기 분기의 Gini 감소량과 비교: 흡연기간이 유리하려면 단기 그룹 내 악성 비율 p가 종양크기로 분기할 때보다 악성이 더 집중되어야 함
(다) 예측 결과
① {단기, 소} → 단기 → 종양크기=소 → [양성] ② {단기, 대} → 단기 → 종양크기=대 → [악성] ③ {장기, 소} → 장기 → [악성] (종양크기 무관)
(라) 루트를 종양크기로 변경 시
[루트] 종양크기? ├─ 소 → [양성] (단순 다수결 또는 추가 분기) └─ 대 → [악성]
예측 결과 달라지는 조합: {단기, 대}
→ 흡연기간 루트 트리: 악성 / 종양크기 루트 트리: 악성 (같음)
→ {장기, 소}: 흡연기간 루트 → 악성 / 종양크기 루트 → 양성(소이므로) : 결과 다름
변형 포인트: 트리 구조를 주고 데이터 역추론 + 분기 순서의 의미 분석. 원본은 데이터→트리 방향이나, 여기서는 트리→데이터 역방향 문제.

선형 SVM — Hard Margin
서포트 벡터 · 연립방정식 · 마진 최대화 · 분류
원본 문제 요약
4포인트 (A, B, C, D), A·C·D가 SV
A(0,3,+1), B(2,2,+1), C(1,0,−1), D(0,1,−1)
SV: A, C, D → 연립방정식 풀이 → w, b, 마진 계산
[응용] SVM — 3-A
⭐⭐ 보통 수치 변형 계산형
P3-A
좌표가 변경된 4개의 포인트로 Hard Margin SVM을 구하시오.
포인트x₁x₂y
A04+1
B33+1
C20−1
D02−1
힌트: w₁, w₂가 양의 정수.
(가)
Support Vector가 될 수 없는 포인트를 예상하고, A, C, D가 SV임을 가정하여 연립방정식을 세우시오.
(나)
연립방정식을 풀어 w = (w₁, w₂)와 b를 구하시오. B가 SV가 아님을 검증하시오.
(다)
결정 경계 방정식과 마진 너비를 계산하시오.
(라)
새로운 포인트 (2, 3)을 분류하시오.
풀이
(가) SV 가정 및 연립방정식
A(0,4,+1), C(2,0,−1), D(0,2,−1) 가 SV라 가정:
A: w·(0,4) + b = 1 → 4w₂ + b = 1 ... ① C: w·(2,0) + b = −1 → 2w₁ + b = −1 ... ② D: w·(0,2) + b = −1 → 2w₂ + b = −1 ... ③
(나) 풀이
①−③: 4w₂ − 2w₂ = 1−(−1) → 2w₂ = 2 → w₂ = 1 ③: 2·1 + b = −1 → b = −3 ②: 2w₁ − 3 = −1 → 2w₁ = 2 → w₁ = 1 ∴ w = (1, 1), b = −3
B 검증:
w·B + b = 1·3 + 1·3 − 3 = 3 > 1 → B는 양성 마진 경계 바깥 → SV 아님 ✓
(다) 결정 경계 및 마진
결정 경계: x₁ + x₂ − 3 = 0 ||w|| = √(1²+1²) = √2 마진 = 2/√2 = √2 ≈ 1.414
(원본 문제와 동일한 w, b, 마진 — 포인트 배치가 유사하게 설계됨) (라) (2, 3) 분류
w·(2,3) + b = 2 + 3 − 3 = 2 > 0 → [+1 클래스]
변형 포인트: 포인트 좌표를 (0,3)→(0,4), (1,0)→(2,0) 등으로 이동. 풀이 구조(연립방정식 설정 → 가감법 풀이 → 검증)는 동일하나 중간 계산값이 달라짐.
원본 문제 요약
A, C, D → SV, B는 SV 아님
w = (w₁, w₂), b 계산 → 마진 = 2/||w||
새 포인트 (1,2) 분류
[응용] SVM — 3-B
⭐⭐⭐ 어려움 역방향 문제 계산형
P3-B
결정 경계와 마진이 주어졌을 때, 서포트 벡터를 역산하시오.
어떤 Hard Margin SVM의 결정 경계와 파라미터가 다음과 같다:
결정 경계: 2x₁ + x₂ − 5 = 0 마진 너비: 2/√5
다음 네 후보 포인트가 주어져 있다:
포인트x₁x₂y
P13+1
Q31+1
R01−1
S2−1−1
(가)
w = (w₁, w₂)와 b를 주어진 결정 경계에서 읽어내고, ||w||를 계산하시오. 마진 = 2/||w||임을 확인하시오.
(나)
네 포인트에 대해 각각 함수 거리(functional margin) yᵢ(w·xᵢ + b)를 계산하고, SV 조건 = 1을 만족하는 포인트를 찾으시오.
(다)
SV가 아닌 포인트는 마진 내부/경계/외부 중 어디에 위치하는가? 그 기하학적 의미를 서술하시오.
(라)
만약 포인트 Q를 (4, 1)로 이동시킨다면, SVM의 결정 경계와 마진이 변하는가? 이유를 SV의 정의로 설명하시오.
풀이
(가) w, b, ||w|| 확인
결정 경계: 2x₁ + x₂ − 5 = 0 w = (2, 1), b = −5 ||w|| = √(4+1) = √5 마진 = 2/√5 ✓
(나) 함수 거리 계산
P(1,3,+1): (+1)·(2·1+3−5) = (+1)·0 = 0 → SV 조건 불만족 Q(3,1,+1): (+1)·(2·3+1−5) = (+1)·2 = 2 → SV 조건 불만족 R(0,1,−1): (−1)·(2·0+1−5) = (−1)·(−4) = 4 → SV 조건 불만족 S(2,−1,−1): (−1)·(2·2−1−5) = (−1)·(−2) = 2 → SV 조건 불만족
→ 함수 거리가 정확히 1인 포인트가 없음. P의 함수 거리 = 0은 결정 경계 위에 위치.
이 데이터셋에서 주어진 결정 경계와 완전히 일치하는 SV 포인트가 없으면, Hard Margin SVM의 해가 이 포인트 집합으로 구성되지 않았음을 의미함.
→ 실제로 SV는 함수 거리 = 1인 포인트여야 하므로, 위 네 포인트 중 SV 없음. (문제의 핵심: 주어진 결정 경계가 이 데이터로부터 나온 것이 아님을 역으로 확인) (다) 마진 외부/경계/내부 위치
SV 조건: yᵢ(w·xᵢ + b) = 1 (마진 경계 위) > 1: 마진 외부 (안전 영역) = 1: 마진 경계 = SV < 1: 마진 내부 (Hard Margin SVM에서는 불가)
P: 함수 거리=0 → 결정 경계 위에 위치 → Hard Margin 조건 위반. (라) Q를 (4,1)로 이동
Q는 SV가 아니었으므로(함수 거리=2 > 1), Q를 이동해도 결정 경계·마진 불변.
SVM은 오직 SV에만 의존하므로, 비-SV 포인트의 이동은 해에 영향 없음.
변형 포인트: w, b → 서포트 벡터 역산 방향. 원본은 포인트 → w, b이지만, 여기서는 결정 경계 → 함수 거리 계산으로 SV 여부를 역검증. SV가 없는 경우를 의도적으로 설계하여 SV의 정의를 깊이 이해하게 함.
원본 문제 요약
Hard Margin 마진 = 2/||w||
새 포인트 (1,2) 분류
힌트: w₁, w₂ 양의 정수
[응용] SVM — 3-C
⭐⭐⭐ 어려움 개념 결합 계산형 + 종합판단형
P3-C
포인트 배치가 바뀌어 SV 조합이 변하는 경우를 분석하시오.
포인트x₁x₂y
A14+1
B02+1
C30−1
D11−1
힌트: w₁, w₂가 양의 정수. B, C, D가 SV라 가정하시오.
(가)
B, C, D가 SV임을 가정하여 연립방정식을 세우고 w, b를 구하시오.
(나)
A가 SV가 아님을 검증하시오. 또한 A가 SV 조건을 만족하려면 A의 x₂ 좌표가 어떤 값이어야 하는지 역산하시오.
(다)
원본 문제(SV: A, C, D)와 이 문제(SV: B, C, D)의 결정 경계가 다른가 같은가? 두 결정 경계를 각각 구하고 비교하시오.
(라)
다음 중 옳은 설명을 모두 고르고 각각의 이유를 서술하시오:
  ① SVM의 결정 경계는 SV 포인트의 위치에만 의존한다.
  ② SV가 아닌 포인트를 제거해도 결정 경계는 불변이다.
  ③ 같은 데이터셋에서 SV 조합이 여러 개 존재할 수 있다.
  ④ 마진이 최대일 때 SVM의 일반화 성능이 가장 낮다.
풀이
(가) B, C, D SV 가정
B(0,2,+1): w·(0,2)+b = 1 → 2w₂+b = 1 ... ① C(3,0,−1): w·(3,0)+b = −1 → 3w₁+b = −1 ... ② D(1,1,−1): w·(1,1)+b = −1 → w₁+w₂+b = −1 ... ③ ①−③: 2w₂ − w₁ − w₂ = 1−(−1) → w₂ − w₁ = 2 ... ④ ②−③: 3w₁ − w₁ − w₂ = −1−(−1) → 2w₁ − w₂ = 0 ... ⑤ ④+⑤: w₁ = 2, w₂ = 4 ②: 3·2 + b = −1 → b = −7 ∴ w = (2, 4), b = −7
(나) A 검증 및 역산
A(1,4,+1): w·(1,4)+b = 2+16−7 = 11 > 1 → SV 아님 ✓ A가 SV 조건 = 1이 되려면: 2·1 + 4·x₂ − 7 = 1 → 4x₂ = 6 → x₂ = 1.5 → A = (1, 1.5)일 때 SV
(다) 원본 vs 이 문제의 결정 경계 비교
원본 문제 (SV: A, C, D): w = (1, 2) (원본 풀이에서 도출), b = -5 → x₁ + 2x₂ − 5 = 0 이 문제 (SV: B, C, D): 2x₁ + 4x₂ − 7 = 0 (또는 정규화 후 다른 방정식) → 두 결정 경계가 다름
→ SV 조합이 달라지면 결정 경계도 달라짐. (라) 선택지 평가
① 옳음: SVM은 SV의 위치로만 결정 경계 결정.
② 옳음: 비-SV 제거 → SV 동일 → 결정 경계 불변.
③ 틀림: Hard Margin SVM 해는 볼록 최적화 → 유일해 → SV 조합도 유일.
④ 틀림: 마진 최대 = 일반화 성능 최고 (SVM의 핵심 원리).
변형 포인트: SV 조합을 A,C,D→B,C,D로 변경. 어떤 포인트가 SV가 되는지에 따라 결정 경계가 달라짐을 확인. 선지 판별형 추가로 SVM 개념 심화.

P4-A
Random Forest와 Bagging의 공통점과 차이점을 비교하고, RF가 Bagging보다 우수한 이유를 랜덤 특성 선택 관점에서 설명하시오.
(가)
Bagging (Bootstrap Aggregating)의 작동 원리를 설명하고, RF와의 공통점을 구체적으로 서술하시오.
(나)
RF가 Bagging과 다른 핵심 요소인 랜덤 특성 선택(Random Feature Subspace)이 트리 간 상관관계(correlation)에 미치는 영향을 설명하시오. 특히 m = √p (특성 수의 제곱근) 설정 이유를 분산 감소 수식과 연결하여 서술하시오.
(다)
다음 시나리오에서 RF와 Bagging의 성능 차이가 어떻게 달라지는지 예측하시오:
— 데이터에 매우 강한 예측력을 가진 특성 1개가 존재하는 경우
— 모든 특성이 균등한 예측력을 가진 경우
채점 기준 및 모범 답안
채점 키워드: Bootstrap 샘플링, 트리 간 상관관계(ρ), 분산 감소 공식, m=√p, 강한 특성 지배

(가) Bagging vs RF 공통점
Bagging: 원본 데이터에서 복원 추출(Bootstrap)로 B개의 훈련셋 생성 → 각각에 완전한 트리 학습 → 다수결/평균으로 예측. RF와 공통점: Bootstrap 샘플링 사용, 앙상블 방식, 분산 감소 목적. (나) 랜덤 특성 선택의 역할
앙상블 분산 공식: Var(ensemble) = ρσ² + (1−ρ)σ²/B ρ: 트리 간 상관계수, σ²: 개별 트리 분산, B: 트리 수 Bagging: 모든 특성 사용 → 강한 특성이 항상 루트 선택 → 트리들이 유사 → ρ 높음 RF: 분기마다 m=√p 특성만 사용 → 강한 특성이 매번 선택 안 됨 → 트리 다양성 ↑ → ρ 감소 → Var(ensemble) 추가 감소
m=√p는 다양성과 예측력 사이의 경험적 최적값. (다) 시나리오별 성능 차이
- 강한 특성 1개 존재: Bagging은 해당 특성이 모든 트리에서 루트 → ρ↑↑ → 분산 감소 효과 제한. RF는 해당 특성이 일부 트리에서 배제 → ρ 감소 → RF가 Bagging보다 우위.
- 균등 특성: 랜덤 선택의 이점 감소 → RF와 Bagging 성능 유사 (ρ 차이 작음).
변형 포인트: 원본은 RF vs 단일 DT 비교. 여기서는 RF vs Bagging 비교로 변경. 더 미세한 차이(랜덤 특성 선택)에 집중하는 심화 서술형.
원본 문제 요약
Soft Margin SVM의 한계 → 커널로 해결, K(x,z) = φ(x)·φ(z)
비선형 분리 불가 데이터 예시 + 커널 트릭 핵심 직관 서술

⭐⭐⭐ 어려움 개념 결합 서술형 · 참/거짓 판별형
P4-B
RBF 커널과 다항식 커널의 특징을 비교하고, 아래 진술의 참/거짓을 판별하시오.
(가)
RBF(Gaussian) 커널 K(x,z) = exp(−γ||x−z||²)이 커널 트릭을 통해 암묵적으로 사상하는 특성 공간의 차원은 몇 차원인가? 이것이 다항식 커널과 어떻게 다른지 서술하시오.
(나)
다음 각 진술이 참인지 거짓인지 판별하고, 배운 이론을 근거로 이유를 제시하시오:
  ① γ 값이 클수록 RBF 커널 SVM의 결정 경계는 더 유연(복잡)해진다.
  ② 커널 SVM은 고차원으로 명시적 변환 후 선형 SVM을 학습하는 것과 계산 비용이 동일하다.
  ③ 원본 공간에서 선형 분리 가능한 데이터는 커널 SVM을 사용할 필요가 없다.
(다)
XOR 문제 (4개 포인트: (0,0), (1,1)은 클래스 −1, (0,1), (1,0)은 클래스 +1)에 대해, 다항식 커널 K(x,z) = (x·z + 1)²를 적용하면 분리 가능해지는 이유를 φ(x)를 명시하여 설명하시오.
채점 기준 및 모범 답안
채점 키워드: 무한차원, 커널 트릭 계산 복잡도, γ와 과적합, XOR φ(x) 명시 (가) RBF 커널의 특성 공간
RBF 커널은 테일러 급수 전개 시 무한 차원 특성 공간으로 사상됨.
다항식 커널 K(x,z)=(x·z+1)^d는 유한 차원(C(d+p, d)차원, p: 원본 특성 수).
RBF: φ: ℝᵖ → ℝ^∞ (무한차원) 다항식(d=2): φ: ℝ² → ℝ⁶ 예: φ(x₁,x₂) = [1, √2x₁, √2x₂, x₁², x₂², √2x₁x₂]
(나) 참/거짓 판별
① 참: γ↑ → 각 포인트의 영향 반경 ↓ → 결정 경계가 각 데이터에 촘촘히 적합 → 복잡도↑ → 과적합 위험.
② 거짓: 명시적 변환 후 선형 SVM의 학습 비용 = O(n × d_high), 커널 SVM의 커널 행렬 계산 = O(n²) + SVM. 고차원일수록 커널 트릭이 훨씬 효율적.
③ 참 (그러나 조건부): 선형 분리 가능이면 선형 SVM으로 충분. 단, 테스트 데이터까지 고려하면 커널 SVM이 더 유연할 수 있음. (다) XOR과 다항식 커널
K(x,z) = (x·z+1)² → φ(x) = [x₁², x₂², √2x₁x₂, √2x₁, √2x₂, 1] 변환 후 포인트: (0,0) → [0,0,0,0,0,1] : 클래스 −1 (1,1) → [1,1,√2,√2,√2,1] : 클래스 −1 (0,1) → [0,1,0,0,√2,1] : 클래스 +1 (1,0) → [1,0,0,√2,0,1] : 클래스 +1 6차원 공간에서 선형 분리 가능 (x₁² + x₂² − √2x₁x₂ 성분 조합으로 분리)
변형 포인트: 원본은 "커널이 해결하는 데이터 유형 서술"이나, 여기서는 RBF vs 다항식 커널 비교 + 참/거짓 판별 + XOR 구체 사례로 변형. 커널 종류를 구체적으로 구분하는 심화 문제.
원본 문제 요약
RF 설명 (세 측면) + 커널 설명 (수식 포함)
두 독립 서술형 문제
[응용] RF — 4-C
⭐⭐⭐ 어려움 개념 결합 + 논술형 논술형 · 자료해석형
P4-C
아래 실험 결과를 해석하고, RF의 앙상블 효과와 SVM 커널 선택 기준을 논술하시오.
실험 설정: 동일한 이진 분류 데이터에 세 가지 모델을 적용한 결과이다.
모델훈련 정확도테스트 정확도비고
단일 DT (depth=∞)100%72%—
RF (100트리)95%88%m=√p
RBF 커널 SVM98%85%γ=0.1, C=1
(가)
단일 DT의 훈련 정확도 100% / 테스트 72%의 차이를 편향-분산 트레이드오프 관점에서 해석하시오.
(나)
RF가 단일 DT 대비 테스트 정확도를 16%p 향상시킨 이유를 Bootstrap Sampling과 Random Feature Selection 두 메커니즘이 각각 기여하는 방식으로 분리하여 서술하시오.
(다)
RBF 커널 SVM(85%)과 RF(88%)의 성능 차이에 대해, γ 값 조정 시 SVM 성능이 어떻게 변화할지 예측하시오. γ를 증가시킬 때와 감소시킬 때의 결과를 각각 서술하시오.
(라)
이 데이터셋에서 최종 모델로 RF를 선택한다면 어떤 한계점이 존재하는가? RF가 취약한 문제 유형을 최소 두 가지 제시하고, 그 대안 모델을 각각 제시하시오.
채점 기준 및 모범 답안
채점 키워드: 과적합, 편향, 분산, 앙상블 효과, γ와 결정경계, RF 한계 (가) 단일 DT 과적합 해석
depth=∞ → 모든 훈련 데이터를 암기(편향=0) → 훈련 100%. 그러나 노이즈까지 학습 → 새 데이터에 일반화 실패(분산↑) → 테스트 72%. 전형적인 고분산·저편향(High Variance, Low Bias) 문제. (나) RF의 16%p 향상 메커니즘
Bootstrap Sampling의 기여: - B개의 서로 다른 훈련셋 생성 - 각 트리가 데이터의 서로 다른 부분집합 학습 - Out-of-Bag 샘플로 자연스러운 검증 효과 - 개별 트리의 고분산을 평균화로 상쇄 Random Feature Selection의 기여: - 강한 특성 지배 방지 → 트리 간 상관관계(ρ) 감소 - Var(RF) = ρσ² + (1−ρ)σ²/B - ρ 감소가 분산 감소에 추가 기여 - 약한 특성도 루트 역할 → 다양한 패턴 포착
(다) γ 조정 효과 예측
γ 증가 → 각 SV의 영향 반경 ↓ → 결정 경계 더 구불구불 → 훈련 정확도↑ but 과적합 위험 → 테스트 정확도 85% → ?% (데이터에 따라 다름. 이미 98% 훈련 정확도이면 과적합 시작 임박 → 테스트 하락 예상).
γ 감소 → 더 부드러운 결정 경계 → 훈련 정확도↓ → 과소적합 가능 → 테스트 정확도 하락 가능. 최적 γ는 Cross-Validation으로 탐색 필요. (라) RF의 한계 및 대안
① 시계열/순차 데이터: RF는 순서 정보 무시 → LSTM, Transformer 적합.
② 매우 고차원 희소 데이터 (텍스트, 유전자): m=√p로도 과다 특성 → 선형 SVM, 로지스틱 회귀 적합.
③ 외삽(Extrapolation): RF는 훈련 범위 밖 예측 불가(앙상블 평균 = 훈련 데이터 범위 내 수렴) → Gradient Boosting, 선형 모델 적합.
변형 포인트: 원본 서술형을 자료(실험 결과 표) 해석형으로 변환. RF와 SVM을 직접 비교하고, RF의 한계까지 논술하는 심화 논술형. 여러 개념의 결합 적용 필요.
