01

디지털-대-아날로그 전환 개요

변조(Modulation)란?

디지털 데이터를 아날로그 신호로 변환하는 과정을 디지털-대-아날로그 변조라 한다. 전화망·무선망처럼 아날로그 매체를 통해 디지털 데이터를 전송할 때 사용한다.

아날로그 매체는 연속적인 주파수 신호를 전송하도록 설계되어 있기 때문에, 디지털 비트열을 직접 실을 수 없다. 변조를 통해 반송파(Carrier)의 특성을 변화시켜 데이터를 실어 보낸다.
ASK
Amplitude Shift Keying

진폭을 변화시켜 0/1을 표현. 주파수·위상은 고정.

FSK
Frequency Shift Keying

주파수를 변화시켜 0/1을 표현. 진폭·위상은 고정.

PSK
Phase Shift Keying

위상을 변화시켜 0/1을 표현. 진폭·주파수는 고정.

QAM
Quadrature Amplitude Modulation

ASK + PSK 조합. 진폭과 위상을 동시에 변화.

02

비트율과 신호율

핵심 용어 정의
비트율 (Bit Rate)
단위: bps (bits per second)
초당 전송되는 비트의 수 N
신호율 (Signal Rate / Baud Rate)
단위: Baud
초당 신호 요소(signal element)의 수 S

• r = 하나의 신호 요소에 담기는 비트 수

• L = 신호 요소의 종류 수 (레벨 수)

• r = log₂ L

변환 공식
S = N × 1/r  [Baud]

N = S × r  [bps]

r = log2 L
직관적 이해
신호율이 낮을수록 대역폭(Bandwidth)을 적게 사용한다. 같은 비트율이라면 r이 클수록(신호 요소 레벨 수가 많을수록) 신호율이 낮아져 효율적이다.
비트율 vs 신호율 관계 그래프

r이 클수록 같은 비트율에서 신호율이 낮아짐 → 대역폭 절약

03

반송파 신호 (Carrier Signal)

반송파란?

아날로그 전송에서 정보 신호의 기반이 되는 고주파 신호. 정보 자체를 담지 않고, 변조를 통해 정보를 실어 나른다.

반송파의 3가지 특성
  • 진폭 (Amplitude) — 신호의 세기
  • 주파수 (Frequency) — 초당 사이클 수
  • 위상 (Phase) — 기준점으로부터의 이동
변조 (Modulation)
반송파의 특성 중 한 가지 이상을 변화시켜 데이터를 표현하는 방식
반송파 신호 파형

A·cos(2πft + φ) — 진폭(A), 주파수(f), 위상(φ)

04

ASK — 진폭편이 변조

ASK (Amplitude Shift Keying) 원리

진폭(Amplitude)을 변화시켜 디지털 비트를 표현한다.

BASK (Binary ASK / OOK)
비트 1 → 높은 진폭, 비트 0 → 진폭 0 (무신호)

• 주파수와 위상은 변하지 않음

• 잡음에 취약 (SNR 영향 큼)

• 광섬유 통신에서 OOK 형태로 사용

• 레벨 수 L = 2이면 r = 1 (비트/신호)

S(t) = Ai · cos(2πfct)
진폭 Aᵢ만 변화, fc 고정
ASK 파형
05

FSK — 주파수편이 변조

FSK 원리

주파수(Frequency)를 변화시켜 비트를 표현한다.

BFSK (Binary FSK)
비트 1 → 높은 주파수 f₁
비트 0 → 낮은 주파수 f₀
  • 진폭과 위상은 변하지 않음
  • ASK보다 잡음에 강함
  • VHF/UHF 무선 통신에 활용
S(t) = A · cos(2πfit)
주파수 fᵢ만 변화, 진폭 A·위상 고정
FSK 파형
06

PSK — 위상편이 변조

PSK 원리

위상(Phase)을 변화시켜 비트를 표현한다.

BPSK (Binary PSK)
비트 1 → 위상 0° (기준)
비트 0 → 위상 180° (반전)
QPSK (Quadrature PSK)
4가지 위상(45°, 135°, 225°, 315°) → 2비트/신호
r = 2, L = 4
  • 진폭과 주파수는 변하지 않음
  • ASK보다 잡음에 강함
  • QPSK: 같은 대역폭에서 2배 비트 전송
  • Wi-Fi, 위성 통신 등에 활용
S(t) = A · cos(2πfct + φi)
위상 φᵢ만 변화, 진폭 A·주파수 fc 고정
PSK 파형 (BPSK)
07

QAM — 구상진폭 변조

QAM 원리

ASK + PSK의 조합. 진폭과 위상을 동시에 변화시킨다.

왜 QAM인가?
ASK 단독보다 잡음에 강하고, PSK보다 더 많은 신호 레벨을 가질 수 있어 높은 비트율을 달성할 수 있다.
QAM 종류레벨(L)r(비트/신호)
4-QAM42
16-QAM164
64-QAM646
256-QAM2568

케이블 TV, LTE/5G, Wi-Fi (802.11) 등에서 광범위하게 사용

16-QAM 성운도 (Constellation)
08

성운그림 (Constellation Diagram)

성운그림이란?

2차원 평면에 신호의 상태를 점으로 표시한 다이어그램.

X축 (수평) — 동위상(In-phase) 반송파
Y축 (수직) — 구상(Quadrature) 반송파
점의 위치 = (진폭, 위상) 조합
  • 원점으로부터의 거리 = 진폭 크기
  • X축으로부터의 각도 = 위상
  • 점의 개수 = 레벨 수 L
ASK(OOK), BPSK, QPSK 비교
09

예제 풀이

⭐ EXAMPLE 5.1 — Slide 5

An analog signal carries 4 bits per signal element. If 1000 signal elements are sent per second, find the bit rate.

풀이

주어진 값: r = 4 S = 1000 Baud N = ?

N = S × r
N = 1000 × 4
N = 4000 bps
핵심 개념: 신호율 S에 신호 하나가 나르는 비트 수 r을 곱하면 비트율 N이 된다.
⭐⭐ EXAMPLE 5.2 — Slide 6

An analog signal has a bit rate of 8000 bps and a baud rate of 1000 baud. How many data elements are carried by each signal element? How many signal elements do we need?

풀이

주어진 값: N = 8000 bps S = 1000 Baud r = ? L = ?

① r 구하기:
N = S × r  →  r = N / S = 8000 / 1000 = 8

② L 구하기:
r = log2 L  →  L = 2r = 28 = 256
해석: 각 신호 요소는 8비트를 나르며, 신호 레벨은 256가지가 필요하다. (256-QAM 수준)
