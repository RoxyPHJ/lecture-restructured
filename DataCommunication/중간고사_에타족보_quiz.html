계산형 주파수·파장 족보 ⭐1번
⭐⭐ 보통

빛의 속도 c = 3×10⁸ m/s를 이용하여, Wi-Fi에서 사용하는 2.4 GHz와 5 GHz 전파의 파장(λ)을 각각 계산하시오. 또한 두 주파수를 파장 길이, 장애물 투과성, 도달 거리 3가지 측면에서 비교하시오.

✓ 정답
2.4 GHz: λ = 0.125 m (12.5 cm)  |  5 GHz: λ = 0.06 m (6 cm)
①λ = c / f  →  λ(2.4G) = 3×10⁸ / 2.4×10⁹ = 0.125 m
②λ(5G) = 3×10⁸ / 5×10⁹ = 0.06 m
③파장 길이: 2.4 GHz > 5 GHz (파장이 길수록 저주파)
④장애물 투과: 2.4 GHz가 유리 (파장이 길어 회절이 잘 됨) → 벽 투과 유리
⑤도달 거리: 2.4 GHz가 더 멀리 도달 / 5 GHz는 속도 빠르지만 거리 짧음

계산형 [응용] 수치변형 주파수·파장
⭐⭐⭐ 어려움

어떤 무선 통신 시스템이 60 GHz 주파수를 사용한다. (c = 3×10⁸ m/s)
① 파장을 구하시오.
② 이 주파수는 2.4 GHz 대비 파장이 몇 배 짧은가?
③ 이 주파수를 실내에서 사용할 때의 단점을 파장 관점에서 설명하시오.

✓ 정답
① λ = 0.005 m (5 mm)  ② 25배 짧음  ③ 장애물 투과 불가, 직진성 강함
①λ = 3×10⁸ / 60×10⁹ = 0.005 m = 5 mm
②0.125 / 0.005 = 25배 (2.4 GHz 파장 0.125 m 대비)
③파장이 매우 짧아 회절 능력이 극히 낮음 → 벽·가구 하나로도 신호 완전 차단. 실내 LOS(가시선) 환경 필수. Wi-Fi 6E에서 60 GHz 활용하지만 같은 방 내에서만 사용.

계산형 SNR 족보 ⭐2번
⭐⭐ 보통

신호 전력이 10 mW, 잡음 전력이 0.01 mW일 때,
① SNR을 구하시오.
② SNRdB를 구하시오. (log₁₀(1000) = 3 이용)

SNR = Signal Power / Noise Power
SNR_dB = 10 × log₁₀(SNR)
✓ 정답
① SNR = 1000  ② SNR_dB = 30 dB
①SNR = 10 / 0.01 = 1000
②SNR_dB = 10 × log₁₀(1000) = 10 × 3 = 30 dB
✓SNR이 높을수록 신호 품질이 좋음. 30 dB는 일반적으로 양호한 통신 품질

계산형 [응용] 역방향 SNR
⭐⭐⭐ 어려움

어떤 채널의 SNR_dB = 36 dB이다. 잡음 전력이 0.001 mW일 때, 신호 전력(mW)을 구하시오.
(hint: 10^3.6 ≈ 3981)

✓ 정답
신호 전력 ≈ 3.981 mW
①SNR_dB = 10 × log₁₀(SNR) → 36 = 10 × log₁₀(SNR) → SNR = 10^3.6 ≈ 3981
②SNR = P_signal / P_noise → P_signal = SNR × P_noise = 3981 × 0.001 ≈ 3.981 mW
✓역방향 문제: SNR_dB → SNR(선형) → 신호전력 순으로 역산. 단위 mW 주의

계산형 샤넌·나이퀴스트 족보 ⭐3번
⭐⭐⭐ 어려움

대역폭 B = 4 kHz, SNR = 255인 채널이 있다.
① 샤넌 공식으로 채널 용량(bps 상한)을 구하시오.
② 나이퀴스트 공식으로 이 채널의 최대 신호 준위 수(L)를 구하시오.
(단, 나이퀴스트 전송률이 샤넌 용량을 넘지 않아야 함)
(log₂(256) = 8)

Shannon: C = B × log₂(1 + SNR)
Nyquist: C = 2B × log₂(L)
✓ 정답
① C = 32,000 bps (32 kbps)  ② L = 16
①C = 4000 × log₂(256) = 4000 × 8 = 32,000 bps
②나이퀴스트: 32,000 = 2 × 4000 × log₂(L) → log₂(L) = 4 → L = 2⁴ = 16
✓샤넌 = 이론적 최대(잡음 고려), 나이퀴스트 = 무잡음 최대(준위 수 고려). 실제 L은 샤넌 한계를 초과 불가

계산형 [응용] 수치변형 샤넌·나이퀴스트
⭐⭐⭐ 어려움

대역폭 B = 1 MHz, 신호 준위 수 L = 8인 채널에서:
① 나이퀴스트 공식으로 최대 데이터 전송률을 구하시오.
② 이 채널의 실제 SNR이 63일 때, 샤넌 용량도 구하고 ①의 결과와 비교하시오.
(log₂(64) = 6, log₂(8) = 3)

✓ 정답
① 6 Mbps  ② 샤넌 6 Mbps → 동일. 나이퀴스트가 샤넌 한계에 딱 맞음
①C_N = 2 × 1×10⁶ × log₂(8) = 2×10⁶ × 3 = 6 Mbps
②C_S = 1×10⁶ × log₂(64) = 10⁶ × 6 = 6 Mbps
✓두 값이 같다 → 이 채널은 나이퀴스트 최대치가 샤넌 상한과 정확히 일치하는 이상적 설계. 실제로는 나이퀴스트 ≤ 샤넌이어야 안전

나이퀴스트 공식(C = 2B log₂L)에서 목표 전송률이 100 Mbps이고 신호 준위 수 L = 4일 때, 필요한 최소 대역폭 B를 구하시오.

✓ 정답
B = 25 MHz
①100×10⁶ = 2 × B × log₂(4) = 2 × B × 2 = 4B

계산형 dBm 족보 ⭐4번

① 전력 100 mW를 dBm으로 변환하시오. (log₁₀(100) = 2)
② dBm = −20 dBm의 전력을 mW로 변환하시오. (10^(−2) = 0.01)

✓dBm은 1 mW 기준. 0 dBm = 1 mW, 10 dBm = 10 mW, 30 dBm = 1 W (10 dB 증가 = 전력 10배)

⭐⭐⭐ 어려움

신호 전력이 +10 dBm, 잡음 전력이 −20 dBm이다.
① 각 전력을 mW로 변환하시오.
② 선형 SNR을 구하시오.

서술+그래프 QPSK 위상 족보 ⭐5번
⭐⭐⭐ 어려움

QPSK에서 2비트 입력 10이 들어왔다. 상위비트(1)는 동위상(I) 반송파 sin(2πft)에, 하위비트(0)는 직교(Q) 반송파 sin(2πft + 90°)에 실린다.
① 상위비트 1 → I 성분의 진폭 부호를 결정하시오 (1→+1, 0→−1 매핑).
② 하위비트 0 → Q 성분의 진폭 부호를 결정하시오.
③ 합산 신호를 수식으로 쓰고, 위상(°)을 구하시오.

✓ 정답
I = +1, Q = −1  →  위상 = −135° (또는 225°)
①상위비트 1 → I 성분 = +1 × sin(2πft)
②하위비트 0 → Q 성분 = −1 × sin(2πft + 90°)
③합산: A sin(2πft + φ)  |  성상도에서 (I=+1, Q=−1) → 4사분면 → φ = arctan(−1/+1) = −45°?
✓주의: QPSK 표준 매핑(Gray code)에서 10은 보통 −135° (3사분면). 교수님 강의 매핑 기준 확인 필수. I=+1, Q=−1이면 −45° 또는 315°.

서술형 회선교환·양자화·부호화 족보 ⭐6번
⭐⭐ 보통

다음 개념을 각각 서술하시오.
① 회선교환(Circuit Switching)이란 무엇인지, 패킷교환과의 차이점 1가지 포함하여 설명하시오.
② 양자화 잡음(Quantization Noise)이란 무엇인지 설명하시오.
③ 회선부호화(Line Coding)가 필요한 이유를 2가지 이상 서술하시오.

✓ 정답
①회선교환: 통신 시작 전에 발신자와 수신자 사이에 전용 물리 회선을 미리 설정하고, 통신이 끝날 때까지 그 회선을 독점 사용하는 방식.

③회선부호화 필요 이유: (a) DC 성분 제거 → 변압기 결합 가능, (b) 동기화(클록 추출) → 수신 측이 비트 경계 파악, (c) 오류 검출 가능, (d) 대역폭 효율화

서술+그래프 맨체스터·비동기직렬 족보 ⭐6번
⭐⭐⭐ 어려움

① 맨체스터(Manchester) 코딩의 규칙을 설명하고, 비트열 1 0 1 1 0의 파형을 아래 SVG 빈칸에 설명으로 답하시오.
② 비동기식 직렬 전송에서 Start bit와 Stop bit가 필요한 이유를 설명하시오.

✓ 정답
①맨체스터 코딩: 각 비트 중간에 전이(transition) 발생.
• 1: 비트 중간에 High → Low (하강 전이)
• 0: 비트 중간에 Low → High (상승 전이)
→ 항상 비트 중간에 전이 있어 클록 동기 추출 가능. DC 성분 없음. 위 파형 참조.
②비동기 직렬 Start/Stop bit: 발신자-수신자 사이에 공통 클록이 없음. Start bit(논리0)로 프레임 시작을 알려 수신 측 클록 동기 시작, Stop bit(논리1)로 프레임 끝과 유휴 상태 알림. 각 문자마다 재동기화.

족보에 주어진 답 "95/96/A/P"의 의미를 해석하여, 각 항목이 무엇을 나타내는지 설명하시오.
(A = ARP로 얻은 다음 홉 MAC, P = 최종 목적지 IP)

A 출발지MAC=95, 목적지MAC=96, 출발지IP=A, 목적지IP=P
B 출발지IP=95, 목적지IP=96, 출발지MAC=A, 목적지MAC=P
C LAN3의 포트번호, 서브넷, 게이트웨이, 프로토콜 번호
D 95=목적지MAC, 96=출발지MAC, A=출발지IP, P=목적지IP
✓ 정답: A
출발지MAC=95, 목적지MAC=96, 출발지IP=A(호스트A), 목적지IP=P(목적지)
✓LAN3 구간 프레임: MAC 헤더(물리주소)에는 해당 링크의 이전 라우터 MAC(95)과 다음 라우터/호스트 MAC(96), IP 헤더(논리주소)에는 종단간 출발지IP(A)와 목적지IP(P)가 유지됨. IP는 종단 유지, MAC은 홉마다 교체.

256-QAM 시스템에서 샘플링 주파수(심볼 전송률)가 10,000 baud일 때,

③ 같은 대역폭에서 16-QAM으로 바꾸면 비트전송률은 어떻게 변하는가?

r = log₂(M)    비트전송률 = Baud × r
✓ 정답

계산형 [응용] 역방향 QAM S값 족보 ⭐QAM d=일때S
⭐⭐⭐ 어려움

어떤 QAM 시스템의 대역폭이 B = 200 kHz이고, 목표 비트 전송률이 C = 1.2 Mbps이다.
나이퀴스트 공식 C = 2B log₂L을 이용하여 필요한 QAM 성상도 심볼 수(S = L²)를 구하시오.
(단, L은 QAM에서 한 축의 준위 수, S = 전체 심볼 수)

✓ 정답
L = 8, S = 64 → 64-QAM
①1.2×10⁶ = 2 × 200×10³ × log₂(L)
②log₂(L) = 1.2×10⁶ / (4×10⁵) = 3 → L = 2³ = 8
③S = L² = 8² = 64 → 64-QAM
✓QAM은 2차원 성상도: I축 L준위 × Q축 L준위 = L² 개 심볼. 이 문제에서 L을 구한 뒤 제곱해야 함

① ② ③ ④

• 채널 대역폭: B = 500 kHz
• 신호전력: +20 dBm, 잡음전력: −10 dBm
• 목표: 64-QAM 방식으로 3 Mbps 전송
① SNR_dB와 선형 SNR을 구하시오.
② 샤넌 용량(이론 최대)을 구하시오. (log₂(1001) ≈ 10)
③ 64-QAM 나이퀴스트 전송률을 구하시오. (log₂(64)=6)
④ 3 Mbps 목표 달성이 가능한지 판단하시오.

③C_N = 2 × 500×10³ × log₂(64) = 10⁶ × 6 = 6 Mbps (나이퀴스트 최대)
④목표 3 Mbps ≤ 샤넌 5 Mbps ✓  |  나이퀴스트 6 Mbps ≥ 3 Mbps ✓ → 설계 가능. 단, 나이퀴스트가 샤넌을 초과(6>5)하므로 실제로는 샤넌 한계 5 Mbps를 넘을 수 없어 3 Mbps는 달성 가능

A 나이퀴스트 샘플링 정리에 따르면, 4 kHz 음성 신호를 디지털로 변환하기 위해 최소 8,000 Hz로 샘플링해야 한다.

C 샤넌 용량 공식 C = B log₂(1+SNR)에서 SNR을 높이면 채널 용량은 선형적으로 증가한다.
D QPSK는 한 심볼이 2비트를 나타내며, BPSK 대비 같은 대역폭에서 2배의 비트 전송률을 달성한다.
